BinkDoFrameAsyncWait

The function BinkDoFrameAsyncWait checks on the status of a background Bink frame decompression.

S32 BinkDoFrameAsyncWait(
  HBINK bink,
  S32 us
);

In Parameters

bink
Specifies the Bink handle.
us
Specifies the microseconds to wait on the decompression.

Returns

Returns one if the background decompression is complete, zero if Bink is still working away.

Discussion

This function checks on the status of a background decompression started with the BinkDoFrameAsync function - that is, it tells you whether the background thread is done or not.

You can pass in a microsecond count to wait on the frame decompression (1,000,000 microseconds in a second, btw), or you can pass in zero to immediately return the "done or not done" status. If you pass in a negative number for the us parameter, then Bink will wait forever for the decompression to finish (won't return until it is done).

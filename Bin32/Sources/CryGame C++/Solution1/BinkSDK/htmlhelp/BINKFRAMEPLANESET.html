BINKFRAMEPLANESET

The structure BINKFRAMEPLANESET is a structure describing a set of planes - one for Y, cR, cB, and alpha.

struct BINKFRAMEPLANESET
{
  BINKPLANE YPlane;
  BINKPLANE cRPlane;
  BINKPLANE cBPlane;
  BINKPLANE APlane;
};

Out Members

YPlane
Specifies the Y plane details.
cRPlane
Specifies the cR plane details.
cBPlane
Specifies the cB plane details.
APlane
Specifies the alpha plane details.

Discussion

This structure used in the BINKFRAMEBUFFERS structure. It described one frame of video in Bink's internal buffers. There are four sets of details here - one for Y, cR, cB, and, optionally, one for the alpha plane.

See the BinkGetFrameBuffersInfo and BinkRegisterFrameBuffers functions for details on using this structure.

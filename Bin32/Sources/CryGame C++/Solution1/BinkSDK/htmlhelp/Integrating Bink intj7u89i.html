Integrating Bink into a Sony PS3 Build Environment

Discussion

To build an application that uses Bink Video, you must include the Bink SDK in the build process. Here are the simple steps to do this:

1) Add the Bink SDK directory to your include and library directories in your project.

2) Include Bink.h in your source files.

3) Link with the BinkPS3.a static library. Bink is directly linked into your application!

4) If you are going to use SPU decompression, you must build an SPU Bink ELF. To do this, link the BinkSPU.a static library with one of the SPU support libraries (BinkSPU_sputhreads.a, BinkSPU_raw.a, or BinkSPU_spurs.a, for example). You can also write your own SPU support libraries to link with BinkSPU.a - see radppu.h and radspu.h for details.

5) Similarily on the PPU side, if you are using SPU decompression, add the matching PPU library for the SPU support that you are using (BinkPPU_sputhreads.a, BinkPPU_raw.a, or BinkPPU_spurs.a, for example) to your link line.

And that's it - you should now be able to use Bink!

Bink SDK 1.9c

BinkSetMixBinVolumes

The function BinkSetMixBinVolumes sets the volumes of the individual mix bins on an Xbox, Xbox 360, Sony PS3 or the Miles Sound System.

void BinkSetMixBinVolumes(
  HBINK bink,
  U32 trackID,
  u32_ptr mix_bins,
  s32_ptr volumes,
  U32 total
);

In Parameters

bink
Specifies the Bink handle.
trackID
Specifies the track ID to change the mix bin volumes for.
mix_bins
Points to an array of the mix bin constants for each of the volumes (pass zero for Miles or PS3).
volumes
Points to an array of the new volume levels.
total
The number of mix bins and volumes in the mix_bins and volumes arrays.

Discussion

This function allows you to set the volume for each of the mix bin destinations on an Xbox, Xbox 360, Sony PS3, or the Miles Sound System.

See the Xbox or Xbox 360 documentation for the mix bin destination values. On Sony PS3 and Miles 7, the volume array is a linear array of speakers - you don't need to pass in any mix bin constants, so just pass in 0 for the mix_bins parameter.

The volume parameter specifies the new volume setting from 0 (silent) to 32768 (full volume).

See the Xbox, PS3, or Miles example programs to see the best way to use this function to enable multi-channel sound playback.

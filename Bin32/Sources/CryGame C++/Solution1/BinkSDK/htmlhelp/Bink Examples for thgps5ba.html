Bink Examples for the Xbox 360

Discussion

The Bink SDK supplies several example programs for Xbox 360. To create these examples in Visual C, just create a new project and include the appropriate C source file (ExamXenon.CPP, for example), and then add the Bink for Xbox 360 static library to the project (BinkXenon.LIB or BinkXenon.LTCG.LIB).

The following is a list of the example files that you can experiment with:

For an example of using a Direct3D pixel shader to display, and background threads to decompress, Bink videos, see the "ExamXenThr.CPP" file. This example uses our simple Direct3D wrapper, so be sure to also add the "BinkTextures.CPP" file to your project.

For an example of using a Direct3D pixel shader to display the Bink video, see the "ExamXenon.CPP" file. This example uses our simple Direct3D wrapper, so be sure to also add the "BinkTextures.CPP" file to your project.

For an example of using a Bink alpha plane with a Direct3D pixel shader, see the "ExamXenA.CPP" file. This example uses our simple Direct3D wrapper, so be sure to also add the "BinkTextures.CPP" file to your project.

Bink SDK 1.9c

Bink Examples for the Nintendo DS

Discussion

The Bink SDK supplies a simple example program for Nintendo DS. To create this example with CodeWarrior, just create a new project and include the appropriate C source file (ExamNDS.C), and then add the Bink for DS static library to the project (BinkNDS.a).

The ExamNDS.C is plays back Bink videos as simply as possible - it can be configured to use 8-bit or 16-bit pixel output with a define at the top of the file. 16-bit video looks much better, though, so we recommend leaving the define alone.

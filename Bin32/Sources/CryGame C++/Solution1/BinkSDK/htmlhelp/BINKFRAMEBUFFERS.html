BINKFRAMEBUFFERS

The structure BINKFRAMEBUFFERS is a structure used as the output of BinkGetFrameBuffersInfo and the input to BinkRegisterFrameBuffers.

struct BINKFRAMEBUFFERS
{
  S32 TotalFrames;
  U32 YABufferWidth;
  U32 YABufferHeight;
  U32 cRcBBufferWidth;
  U32 cRcBBufferHeight;
  U32 FrameNum;
  BINKFRAMEPLANESET Frames;
};

Out Members

TotalFrames
Number of plane sets used by this movie (1 for movies with all key frames, 2 for any other).
YABufferWidth
Specifies the width of the Y and alpha planes.
YABufferHeight
Specifies the height of the Y and alpha planes.
cRcBBufferWidth
Specifies the width of the cR and cB planes.
cRcBBufferHeight
Specifies the height of the cR and cB planes.
FrameNum
Specifies current active plane set.
Frames
An array of plane sets that is TotalFrames long.

Discussion

This structure is used when you want to manage the memory used by the internal Bink frame buffer sets. Bink buffers two frames of video internally, with 3 or 4 (with alpha) planes per frame of video. This structure is used twice in an application when it is used at all - once with BinkGetFrameBuffersInfo to fill out the structure initially, and once again with BinkRegisterFrameBuffers once you have allocated the memory pointers.

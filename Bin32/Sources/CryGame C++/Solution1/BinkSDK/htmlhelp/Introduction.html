First off, welcome to the Bink SDK, and thanks for your purchase!

This documentation will describe the Bink SDK for all of our platforms. We spend quite a bit of time working on our docs and hope they get you up and running fast. Please give us feedback on what worked for you and what didn't. Just as with the Bink API itself, we strive for continual improvement in our documentation as well. Your feedback is crucial in this regard!

Also note that this manual actively updated, so be sure to check out these docs whenever you receive a new Bink version, so that you don't miss anything!

The "Bink Overview" chapter of this manual is designed to give you a broad overview of Bink. This is the chapter to read when you're just starting out with our video APIs.

The next ten or so sections of the manual discuss Bink on each of the many platforms we support: Win32, Win64, Sony Playstation 3, Sony PSP, Sony Playstation 2, Xbox 360, Xbox, Nintendo GameCube and Wii, MacOS and, finally, Linux.

Then, we'll wrap up with the reference section of our documentation. These reference chapters will describe each and every function in both the Bink API and our Windows and MacOS blitting API (the BinkBuffer API).

Again, please send feedback to me as you use Bink - what confused you, what you liked, what you hated, what you'd like to see in future versions, etc. Your feedback is what will keep Bink vital, fun to use, and powerful into the future!

Thanks again,

Jeff Roberts - RAD Game Tools - E-mail.

RAD_IOPGetFileHandleToLSN

The function RAD_IOPGetFileHandleToLSN creates a file handle that directly address a Logical Sector Number on a CD or DVD.

U32 RAD_IOPGetFileHandleToLSN(
  U32 LSN
);

In Parameters

LSN
Specifies the logical sector number on the CD or DVD.

Returns

File handle to the SPU memory.

Discussion

This function creates a file handle that can be used to read asynchronously from a specific sector off a CD or DVD.

You don't have to close this type of file handle.

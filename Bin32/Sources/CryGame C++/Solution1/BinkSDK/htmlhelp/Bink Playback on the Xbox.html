Bink Playback on the Xbox

Discussion

The Xbox has great support for threads, so the Xbox Bink playback loop is typically very simple. Bink handles timing with the BinkWait function, so we just need to check this function while we're playing a video.

So, a simple example of Xbox playback would look something like this: 


while ( 1 ) 
{
  // decompress a frame 
  BinkDoFrame( bink );


  // skip frames if we need to...
  while ( BinkShouldSkip( bink )
  {
    BinkNextFrame( bink );
    BinkDoFrame( Bink );
  }


  // draw the video textures here, but don't show the frame yet
  Draw_Bink_textures( ... );


  // check input, process other tasks - whatever
  while ( BinkWait( bink ) )
    do_idle();


  // flip the buffers to show the video frame
  d3d_device->Present();


  // is the video done?
  if ( bink->FrameNum == bink->Frames )
    break;

               
  // move to the next frame
  BinkNextFrame( bink );  
}

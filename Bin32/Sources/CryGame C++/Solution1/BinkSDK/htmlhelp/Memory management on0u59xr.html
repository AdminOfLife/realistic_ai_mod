Memory management on the Sony PS3

Discussion

By default, Bink handles memory management by calling the Sony PS3 C runtime library memory allocation functions directly (malloc and free). You can, however, replace the Bink allocation and freeing functions with the BinkSetMemory function.

You can see how much memory is being used and what the peak use was with the BinkGetSummary function.

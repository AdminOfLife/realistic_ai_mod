Bink Examples for the Nintendo GameCube

Discussion

The Bink SDK supplies several example programs for Nintendo GameCube. To create these examples with SN Systems or CodeWarrior, just create a new project and include the appropriate C source file (ExamNGCbb.C or ExamNGCtex.C), and then add the Bink for Nintendo GameCube static library to the project (BinkNGC.a).

The following is a list of the example files that you can experiment with:

For an example of using Bink to display a video directly to the GameCubes's YUV back buffer, see the "ExamNGCbb.C" file. This example is extremely simple because it doesn't use the GPU to render the video - it just blasts it right to the back buffer. It's also very fast, because Bink doesn't have to convert out of YUV format.

For an example of using the Nintendo GameCube GPU hardware to render directly from Bink's internal YUV buffers, see the "ExamNGCtex.C" file. This example uses our simple 3D wrapper, so be sure to also add the "NGCtextures.C" file to your project. It will render either normal or alpha Bink videos.

All of these examples also use a simple file to setup the GameCube for use, so you'll also need to include "NGCDemo.c" in your sample projects.

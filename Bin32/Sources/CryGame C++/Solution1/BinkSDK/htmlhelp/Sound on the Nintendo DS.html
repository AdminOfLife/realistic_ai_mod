Sound on the Nintendo DS

Discussion

For sound support on the Nintendo DS, Bink talks to directly to the sound driver, so integrating is painless.

To use Bink's built-in DS audio support, you simply have to call BinkSoundUseNDSSound before you call BinkOpen. Once you make this call Bink will handle all of the audio automatically.

For best decompression performance, we recommend using an audio playback rate of 16 Khz or 22 Khz. Also, stereo playback takes twice the CPU power, so try to use mono if you can manage it.

And that's it - Bink takes care of everything else!

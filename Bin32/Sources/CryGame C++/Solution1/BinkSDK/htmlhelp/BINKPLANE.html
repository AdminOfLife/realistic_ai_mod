BINKPLANE

The structure BINKPLANE is a structure describing a single two-dimensional plane.

struct BINKPLANE
{
  S32 Allocate;
  VOID_PTR Buffer;
  U32 BufferPitch;
};

Out Members

Allocate
Specifies whether you need to allocate this plane.
Buffer
Specifies the address to decompress to (should be aligned on a 16-byte boundary).
BufferPitch
Specifies the pitch of the buffer in bytes from one scanline to the next.

Discussion

This the lowest level of detail of the BINKFRAMEBUFFERS structure. There is one of these structure describing each of the Bink video planes - one for Y, cR, cB, and, optionally, alpha.

The Allocate field tells you whether you need to allocate this plane at all. For example, on videos without an alpha plane, the Allocate field would be zero for the APlane structure inside the BINKFRAMEPLANESET structure.

Note that the Buffer memory pointer should be aligned on a 16-byte boundary.

See the BinkGetFrameBuffersInfo and BinkRegisterFrameBuffers functions for details on using this structure.

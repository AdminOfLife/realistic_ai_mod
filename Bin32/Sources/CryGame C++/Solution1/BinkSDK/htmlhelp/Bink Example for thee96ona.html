Bink SDK 1.9c

Bink Example for the Sony PSP

Discussion

The Bink SDK supplies an complete example program for the Sony PSP. To create this example, just create a new project and include the C source files (ExamPSP.C, PageFlip.C and BinkGuTextures.C), and then add the Bink for Sony PSP static library to the project (BinkPSP.a).

This example shows how to draw normal or alpha video using the PSP's 3D hardware.

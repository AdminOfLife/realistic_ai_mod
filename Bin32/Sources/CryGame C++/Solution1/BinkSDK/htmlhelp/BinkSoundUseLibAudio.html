BinkSoundUseLibAudio

The function BinkSoundUseLibAudio tells Bink to use LibAudio as its sound output system on the Sony PS3.

S32 BinkSoundUseLibAudio(
  U32 number_of_speakers
);

In Parameters

number_of_speakers
Specifies the number of speakers to initialize LibAudio to use.

Returns

Returns whether Bink successfully switched to LibAudio as its output layer (non-zero for success, zero for failure).

Discussion

This function (macro actually) tells Bink to use the Sony LibAudio system API for its audio output.

Bink on the PS3 supports up to 7.1 audio output (and any format up to 7.1, like 5.1, or just stereo). This function tells Bink how many of speakers to initialize LibAudio with. This value will usually be retrieved with the LibAudio function, "cellAudioOutGetDeviceInfo". Make sure to also call "cellAudioOutConfigure" to tell the PS3 hardware how many speakers to dump out as well. See the "ExamPS3.cpp" file for details.

This function is global - you only need to call it once. You must, however, call it before you call BinkOpen for the first time, or the function will fail.

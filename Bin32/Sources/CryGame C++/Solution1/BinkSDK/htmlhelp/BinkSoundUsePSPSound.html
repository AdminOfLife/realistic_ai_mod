BinkSoundUsePSPSound

The function BinkSoundUsePSPSound tells Bink to use libwave or simple audio as its sound output system on the Sony PSP.

S32 BinkSoundUsePSPSound(
  U32 channel
);

In Parameters

channel
Specifies the sound channel to use.

Returns

Returns whether Bink successfully switched to PSP sound as its output layer (non-zero for success, zero for failure).

Discussion

This function (macro actually) tells Bink to use either the Sony libwave or the Sony simple audio system API for its audio output.

The channel parameter should be 0, 1, or 2 to tell Bink to use one of the 3 libwave channels, or -1 to tell Bink to use simple audio.

The simple audio API only allows one client to use it at a time, so if you use simple audio, then you must stop your game audio first. To do that, first call sceAudioOutput2Release, then play your movie, and then call sceAudioOutput2Reserve again after video playback is done.

This function is global - you only need to call it once. You must, however, call it before you call BinkOpen for the first time, or the function will fail.

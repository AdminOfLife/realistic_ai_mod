Bink Examples for the Sony PS3

Discussion

The Bink SDK supplies several example programs for PS3. To build these examples, just create a new project and include the appropriate C source file (ExamPS3.CPP, for example), and then add the Bink for Sony PS3 static library to the project (BinkPS3.a).

The following is a list of the example files that you can experiment with:

For an example of using SPU decompression and a libGcm fragment shader to display the Bink video, see the "ExamSPU.CPP" file. This example uses our simple libGcm wrapper, so be sure to also add the "GCMTextures.CPP" file to your project.

For an example of using a libGcm fragment shader to display the Bink video, see the "ExamGCM.CPP" file. This example uses our simple libGcm wrapper, so be sure to also add the "GCMTextures.CPP" file to your project.

For an example of using an PSGL fragment shader to display the Bink video, see the "ExamPS3.CPP" file. This example uses our simple OpenGL wrapper, so be sure to also add the "PSGLTextures.CPP" file to your project.

For an example of using a Bink alpha plane with a PSGL fragment shader, see the "ExamPS3A.CPP" file. This example uses our simple OpenGL wrapper, so be sure to also add the "PSGLTextures.CPP" file to your project.

BinkSoundUseWaveOut

The function BinkSoundUseWaveOut tells Bink to use the deprecated waveOut audio interface as its sound output system.

S32 BinkSoundUseWaveOut();

Returns

Returns whether Bink successfully switched to waveOut its output layer (non-zero for success, zero for failure).

Discussion

This function (macro actually) tells Bink to use the waveOut API for its audio output. WaveOut has been deprecated for a long time, so you generally won't need to use this function. But, if you are supporting really old hardware (or Windows 95), then it's here for you.

This function is global - you only need to call it once. You must, however, call it before you call BinkOpen for the first time, or the function will fail.

Bink SDK 1.9c

Bink Examples for the Nintendo Wii

Discussion

The Bink SDK supplies several example programs for Nintendo Wii. To create these examples with CodeWarrior, just create a new project and include the appropriate C source file (ExamWiiBB.C or ExamWiiTex.C), and then add the Bink for Nintendo Wii static library to the project (BinkWii.a).

The following is a list of the example files that you can experiment with:

For an example of using Bink to display a video directly to the Wii's YUV back buffer, see the "ExamWiiBB.C" file. This example is extremely simple because it doesn't use the GPU to render the video - it just blasts it right to the back buffer. It's also very fast, because Bink doesn't have to convert out of YUV format.

For an example of using the Nintendo Wii GPU hardware to render directly from Bink's internal YUV buffers, see the "ExamWiiTex.C" file. This example uses our simple 3D wrapper, so be sure to also add the "WiiTextures.C" file to your project. It will render either normal or alpha Bink videos.

All of these examples also use a simple file to setup the Wii for use, so you'll also need to include "WiiDemo.c" in your sample projects.

The Nintendo DS Platform

Discussion

Bink now runs on Nintendo's fun little DS handheld! The DS is pretty fun to work on - nice and old-school. The CPU is a bit on the slow side, so we had to do a fair amount of optimization to get Bink running smoothly, but it now runs great.

Bink is extremely easy to use and integrate under this platform. This chapter will discuss everything you need to know!

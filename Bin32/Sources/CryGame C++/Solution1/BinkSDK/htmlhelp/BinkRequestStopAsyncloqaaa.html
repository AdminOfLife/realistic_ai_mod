Bink SDK 1.9c

BinkRequestStopAsyncThread

The function BinkRequestStopAsyncThread starts the process of shutting down a background Bink thread.

S32 BinkRequestStopAsyncThread(
  U32 thread_index
);

In Parameters

thread_index
Specifies the thread index to shutdown.

Returns

Returns one on success, zero if there was an error.

Discussion

This function requests that Bink being closing one of the background threads. This function doesn't wait for the function to exit - you must call BinkWaitStopAsyncThread to wait until a Bink thread has completely closed.

You must call this function for any thread index that you called BinkStartAsyncThread on.

The Win64 Platform

Discussion

Bink now runs great on the pleasently familar Win64 platform. Win64 is pretty much just like developing on Win32 - they did a great job with API compatibility.

Bink now supplies background thread decompression on Win64. We recommend everyone switch to threaded playback on Win64 - even on single core CPUs. It will provide much smoother playback.

Bink is extremely easy to use and integrate under this platform. This chapter will discuss everything you need to know!

BINK_BG_CONTROL_FLAGS

The basic type BINK_BG_CONTROL_FLAGS Specifies the IO state to switch to.

Fundamental Values

 Value  Description
 BINKBGIOSUSPEND Suspends all background IOs. If the internal IO buffer is emptied, then Bink will still do IO in the foreground (in BinkNextFrame).
 BINKBGIORESUME Resumes background IOs.
 BINKBGIOWAIT You can OR this flag with the other flags to spin until the IO is suspended or resumed. If you use this flag with BINKBGIORESUME when the video is already resumed, then it will perform an IO immediately. This is useful when you want to force IO in the foreground.

Discussion

These flags are used with the BinkControlBackgroundIO function to control Bink's background IO system.

The Xbox 360 Platform

Discussion

Bink was one of the very first middleware packages available for the Xbox 360 game console. The Xbox 360 is really fast - you'll be able to make play beautiful videos on it. Microsoft requires HDTV resolution videos (at least 1280x720p), so you'll love Bink's speed.

We did a lot of optimizations for this platform. It was kind of a combination of the Mac assembly code, with the Xbox system support, with the Direct3D 9 API from Windows. And then we did a bunch of new pixel shader work to make it even faster!

Bink is extremely easy to use and integrate under this platform. This chapter will discuss everything you need to know!

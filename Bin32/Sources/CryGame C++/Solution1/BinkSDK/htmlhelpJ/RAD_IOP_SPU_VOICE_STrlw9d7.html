Bink SDK 1.9c

RAD_IOP_SPU_VOICE_STATE_CALLBACK

関数 RAD_IOP_SPU_VOICE_STATE_CALLBACK は、非同期のクエリが完了したときに呼び出される関数をポイントします。

void RAD_IOP_SPU_VOICE_STATE_CALLBACK(
  RAD_IOP_SPU_VOICE_STATE const *query_data
);

入力パラメータ

query_data
RAD_IOPQueueAsyncSPUVoiceState を呼び出したときの構造体へのポインタを指定します。

説明

この関数は、クエリの完了時に (インタラプト レベルで) 1 回呼び出されます。この関数の中に作成するコードは、必ずインタラプト セーフにします。

このコールバックは成功と失敗のどちらの場合も実行されるため、status フィールドの値を確認して、クエリが実際に正常に行われたかどうかを判別してください。


グループ: ソニー PS2 用 RAD_IOP SPU ハードウェア音声 API
関連する関数: RAD_IOPQueueAsyncSPUVoiceState
関連する基本データ型: void
関連する構造体: RAD_IOP_SPU_VOICE_STATE

Bink SDK 1.9c

BinkSetVolume

BinkSetVolume 関数は、Bink のサウンド トラックの音量を設定します。

void BinkSetVolume(
  HBINK bink,
  U32 trackID,
  S32 volume
);

入力パラメータ

bink
Bink ハンドルを指定します。
trackID
音量を変更するトラック ID を指定します。
volume
音量の新しいレベルを指定します。

説明

この関数は、現在再生している Bink のサウンド トラックの音量を設定します。

volume パラメータには、音量の新しい設定を 0 (無音) から 32768 (標準)、65536 (最大に増幅) の範囲で指定します。

ほとんどのサウンド システムは (Miles と DirectSound のどちらも) 増幅を行わないことに注意してください。その場合、32768 以上の音量は 32768 と同じです。

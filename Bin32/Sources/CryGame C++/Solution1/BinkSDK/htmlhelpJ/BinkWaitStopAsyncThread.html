Bink SDK 1.9c

BinkWaitStopAsyncThread

The function BinkWaitStopAsyncThread waits until a Bink background threads closes down.

S32 BinkWaitStopAsyncThread(
  U32 thread_index
);

In Parameters

thread_index
Specifies the thread index to wait on.

Returns

Returns one on success, zero if there was an error.

Discussion

This function waits on a previous shutdown request made by BinkRequestStopAsyncThread.

When shutting down multiple Bink threads, you should call BinkRequestStopAsyncThread on all of them first, and then call BinkWaitStopAsyncThread to make sure each of them is finished.

You must call this function for any thread index that you called BinkStartAsyncThread on.

RAD_IOP_TIMER_DATA

構造体 RAD_IOP_TIMER_DATA は、タイマー インタラプト コールバックの呼び出しに使用する構造体です。

struct RAD_IOP_TIMER_DATA
{
  RAD_IOP_TIMERCALLBACK callback;
  RAD_IOP_STATUS_VALUES status;
};

入力メンバ

callback
タイマー コールバック関数のアドレスを指定します。

出力メンバ

status
タイマーの現在のステータスを示します。

説明

この構造体は、タイマー インタラプト コールバックを呼び出すデータを格納します。これは、タイマーの登録が解除されるまで有効にしておく必要があります。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPPauseTimer, RAD_IOPRegisterTimerCallback, RAD_IOPStartTimer, RAD_IOPUnregisterTimerCallback, RAD_IOP_TIMERCALLBACK
関連する基本データ型: RAD_IOP_STATUS_VALUES

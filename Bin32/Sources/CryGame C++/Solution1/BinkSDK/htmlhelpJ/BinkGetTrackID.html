Bink SDK 1.9c

BinkGetTrackID

BinkGetTrackID 関数は、指定されたトラック インデックスのトラック ID を返します。

U32 BinkGetTrackID(
  HBINK bink,
  U32 track_index
);

入力パラメータ

bink
Bink ハンドルを指定します。
track_index
クエリするトラック インデックスを指定します。

戻り値

指定したトラック インデックスのトラック ID を返します。

説明

Bink のトラック インデックスの範囲は、常に 0 から (ファイルにあるオーディオ トラックの最大数 -1) までです。この関数を使用すると、Bink ファイルをミックスしたユーザーが入力した元のトラック ID を取得できます。

Bink ファイル内の最大のトラック数は、BINK 構造体の NumTracks フィールドで指定されます。したがって、NumTracks が 2 の場合、有効なトラック インデックスは 0 か 1 です。

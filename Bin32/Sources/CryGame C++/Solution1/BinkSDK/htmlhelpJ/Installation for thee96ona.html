ソニー PSP へのインストール

説明

Bink SDK をインストールするには、当社の FTP サイトからダウンロードしたインストーラ (BinkPSP.exe) を実行します。

大部分のインストール オプションは自動的に選択されます。インストール ディレクトリなどのパラメータを変更しないかぎり、ユーザーが入力する必要はほとんどありません。

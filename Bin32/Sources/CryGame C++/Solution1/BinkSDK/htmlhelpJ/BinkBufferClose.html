Bink SDK 1.9c

BinkBufferClose

BinkBufferClose 関数は、指定された BinkBuffer ハンドルを閉じます。

void BinkBufferClose(
  HBINKBUFFER bink_buf
);

入力パラメータ

bink_buf
閉じる BinkBuffer ハンドルを指定します。

説明

この関数は単に BinkBuffer を閉じて、関連するすべてのメモリとシステム リソースを解放するだけです。開いたすべての BinkBuffer を閉じないと、システム リソースまたはメモリがリークすることがあります。


グループ: BinkBuffer API
関連項目: Open と Close
関連する関数: BinkBufferOpen
関連する基本データ型: HBINKBUFFER, void

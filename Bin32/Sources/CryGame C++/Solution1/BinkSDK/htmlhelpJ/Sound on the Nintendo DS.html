ニンテンドー DS のサウンド

説明

ニンテンドー DS でサウンドをサポートするために、Bink はサウンド ドライバを直接制御するため、統合は簡単です。

Bink に組み込まれた DS オーディオ サポートを使用するには、BinkOpen を呼び出す前に BinkSoundUseNDSSound を呼び出します。この呼び出しを行うと、Bink がすべてのオーディオを自動的に処理します。

解凍の効率を最大限に高めるには、16 kHz または 22 kHz のオーディオ再生レートを使用することをお勧めします。また、ステレオ再生は CPU の能力を 2 倍使用するため、モノラルを使用できる場合はモノラルを使用してみてください。

これで完了です。他の部分はすべて Bink が処理します。

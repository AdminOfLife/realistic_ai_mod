Bink SDK 1.9c

BinkShouldSkip

BinkShouldSkip 関数は、フレームまたはブリットをスキップする必要があるほどビデオがオーディオより遅れているかどうかを報告します。

S32 BinkShouldSkip(
  HBINK bink
);

入力パラメータ

bink
Bink ハンドルを指定します。

戻り値

フレームまたはブリットをスキップする場合は 0 以外を返します。そうでない場合は 0 を返します。

説明

この関数を使用して、フレームまたはブリットをスキップするかどうかを決定できます。

テクスチャまたは DirectDraw サーフェスをロックする前に、BinkCopyToBuffer でフレームをスキップする場合に便利です。

また、BinkCopyToBuffer を呼び出すことがないため、輝度テクスチャで BinkRegisterFrameBuffers 関数を使用する場合にも便利です。


グループ: Bink API
関連する関数: BinkCopyToBuffer, BinkRegisterFrameBuffers
関連する基本データ型: HBINK, S32

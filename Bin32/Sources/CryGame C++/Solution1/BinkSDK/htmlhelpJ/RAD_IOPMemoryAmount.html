RAD_IOPMemoryAmount

関数 RAD_IOPMemoryAmount は、RAD_IOP が必要とする IOP メモリの大きさをバイト単位で返します。

U32 RAD_IOPMemoryAmount(
  SOUND_CORE_FLAGS sound_flags
);

入力パラメータ

sound_flags
RAD_IOPStartUp 呼び出しで使用するサウンド フラグを指定します。

戻り値

割り当てる必要がある IOP メモリの大きさを返します。

説明

この関数は、RAD_IOP がフラグのために必要とする IOP メモリの大きさをユーザーに通知します。ユーザーは、通常は sceSifAllocIopHeap を呼び出してこのメモリを割り当て、メモリのアドレスを RAD_IOPStartUp に渡します。このメモリを解放するときは、あらかじめ RAD_IOPShutDown を呼び出す必要があります。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPMemoryAmount, RAD_IOPShutDown, RAD_IOPStartUp
関連する基本データ型: SOUND_CORE_FLAGS, U32

Bink SDK 1.9c

BinkCloseTrack

BinkCloseTrack 関数は、低レベルのオーディオ トラック デコーダのハンドルを閉じます。

void BinkCloseTrack(
  HBINKTRACK bink_track
);

入力パラメータ

bink_track
閉じるハンドルを指定します。

説明

この関数は、BinkOpenTrack で開かれた Bink のオーディオ トラックのハンドルを閉じます。

この関数は、ほとんど使用されません。通常は、オーディオの再生は Bink によって完全に自動的に行われます。


グループ: Bink API
関連する関数: BinkCloseTrack, BinkGetTrackData, BinkGetTrackID, BinkOpenTrack
関連する基本データ型: HBINKTRACK, void

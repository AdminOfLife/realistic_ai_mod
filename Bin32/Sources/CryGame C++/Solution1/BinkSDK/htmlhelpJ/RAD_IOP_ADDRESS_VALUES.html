RAD_IOP_ADDRESS_VALUES

基本データ型 RAD_IOP_ADDRESS_VALUES は、RAD_IOP 非同期 I/O 関数用に読み込むメモリの種類を指定します。

基本的な値

 値  説明
 RAD_IOP_EE_ADDRESS 読み込み先のポインタは通常の EE メモリ アドレスです。
 RAD_IOP_IOP_ADDRESS 読み込み先のポインタは IOP メモリ アドレスです。
 RAD_IOP_SPU_ADDRESS 読み込み先のポインタは SPU メモリ アドレスです。SPU に読み取る場合は、常に 64 バイトの倍数を読み取ります。読み込み先のアドレスは 16 バイト境界で揃えられている必要があります。

説明

これらのフラグは、RAD_IOP の非同期読み取り関数で使用されます。これを使用して、読み取ったデータを配置する場所を制御します。

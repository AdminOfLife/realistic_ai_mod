Bink SDK 1.9c

RAD_IOPUnregisterTimerCallback

関数 RAD_IOPUnregisterTimerCallback は、タイマー コールバックの登録を解除します。

S32 RAD_IOPUnregisterTimerCallback(
  RAD_IOP_TIMER_DATA *timer_data
);

出力パラメータ

timer_data
登録解除するタイマー構造体を指定します。

戻り値

正常ならば 0 以外の値、失敗の場合は 0 を返します。

説明

この関数は、バックグラウンド タイマーを停止し、タイマー コールバック構造体の登録を解除します。

この関数は、必ず RAD_IOPRegisterTimerCallback に渡されたメモリを解放する前に呼び出します。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPRegisterSoundCallback, RAD_IOPRegisterTimerCallback
関連する基本データ型: S32
関連する構造体: RAD_IOP_TIMER_DATA

RAD_IOPSetSPUVoiceADSR

関数 RAD_IOPSetSPUVoiceADSR は、1 つ以上の SPU 音声のエンベロープを設定します。

S32 RAD_IOPSetSPUVoiceADSR(
  U32 core0_voice_mask,
  U32 core1_voice_mask,
  U32 adsr1,
  U32 adsr2
);

入力パラメータ

core0_voice_mask
設定するコア 0 の音声のビット マスクを指定します。
core1_voice_mask
設定するコア 1 の音声のビット マスクを指定します。
adsr1
最初のエンベロープ ビット パターンを指定します。
adsr2
2 番目のエンベロープ ビット パターンを指定します。

戻り値

正常ならば 0 以外の値、失敗の場合は 0 を返します。

説明

この関数は、1 つ以上の SPU 音声の音量エンベロープを設定するように SPU に指示します。

エンベロープ ビット パターンはかなり複雑で、基本的に 2 つの 32 ビット ワードにエンコードされる大きなテーブル インデックス セットです。音声エンベロープを設定する場合は、ソニーのドキュメントを直接参照する必要があります。

瞬時のオン、完全な持続、瞬時のオフという単純なエンベロープの場合は、adsr1 に 0x000f、adsr2 に 0x1fc0 を設定します。

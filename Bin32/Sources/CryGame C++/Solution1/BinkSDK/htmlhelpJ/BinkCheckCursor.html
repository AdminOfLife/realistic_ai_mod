Bink SDK 1.9c

BinkCheckCursor

関数 BinkCheckCursor は、現在のマウスの位置が指定された長方形のウィンドウの内側であるかどうかをチェックし、内側の場合はマウスを表示しないようにします。

S32 BinkCheckCursor(
  OS_WND window,
  S32 check_x,
  S32 check_y,
  U32 width,
  U32 height
);

入力パラメータ

window
check_x と check_y パラメータが関連するウィンドウのハンドル (Win32 の場合) またはウィンドウへのポインタ (MacOS の場合) を指定します。このパラメータには、NULL 値を渡すことができます。
check_x
テスト長方形の X (左) 座標を指定します。
check_y
テスト長方形の Y (上) 座標を指定します。
width
テスト長方形の幅を指定します。
height
テスト長方形の高さを指定します。

戻り値

カーソルを非表示にする必要があった回数の合計を返します (BinkRestoreCursor に渡されます)。

説明

この関数は、現在のマウスの位置が指定されたウィンドウの長方形内であるかどうかを判別し、内側であれば非表示にします。これにより、マウスを動かしても、カーソルを表示していると残る軌跡が表示されないようにします。

この関数は、プライマリ サーフェスをロックする前に呼び出す必要があります。そうしないと、カーソルは非表示になりません。この関数の使用例については、BinkIsSoftwareCursor 関数を参照してください。

この関数は、マウス カーソルを元の状態に復元するために BinkRestoreCursor 関数に渡す必要があるカウント値を返します。

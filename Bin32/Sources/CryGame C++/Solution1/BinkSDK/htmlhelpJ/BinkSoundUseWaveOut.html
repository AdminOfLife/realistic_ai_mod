Bink SDK 1.9c

BinkSoundUseWaveOut

BinkSoundUseWaveOut 関数は、非推奨の waveOut オーディオ インターフェイスをサウンド出力システムとして使用するように Bink に指示します。

S32 BinkSoundUseWaveOut();

戻り値

Bink の出力レイヤが waveOut に切り替わったかどうかを返します (正常に終了した場合は 0 以外、失敗した場合は 0)。

説明

この関数 (実際はマクロ) は、オーディオ出力として waveOut API を使用するように Bink に指示します。waveOut は、長い間、非推奨とされてきています。したがって、通常、この関数を使用する必要はありません。ただし、かなり古いハードウェア (または Windows 95) をサポートする場合は、waveOut を使用できます。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前に呼び出さなければ、この関数は失敗します。


グループ: Bink のオーディオ システムのインターフェイス関数
関連項目: Win32 でのサウンド, Win64 でのサウンド
関連する関数: BinkOpen, BinkSetSoundSystem, BinkSoundUseAX, BinkSoundUseDirectSound, BinkSoundUseLibAudio, BinkSoundUseMiles, BinkSoundUseMusyX, BinkSoundUseNDSSound, BinkSoundUsePSPSound, BinkSoundUseRAD_IOP
関連する基本データ型: S32

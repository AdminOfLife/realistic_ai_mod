ソニー PSP のサウンド

説明

ソニー PSP でサウンドをサポートするために、Bink は libwave とシンプル オーディオ API の 2 つへのインターフェイスを提供します。

libwave を使用すると、ゲームがオーディオを再生しているときに、同時にオーディオ トラックを再生できます (ただし、少し多く CPU を使用します)。シンプル オーディオ API はハードウェアを直接制御するため、少し高速ですが、Bink の再生中に他のオーディオを再生することはできません。

Bink では、すべてのオーディオをあらかじめ 1 つにミックスしてから、1 つのオーディオ チャンネルに渡します。たとえば、複数のサウンド トラックを持つ複数の Bink を再生できますが、その場合も使用する出力サウンド チャンネルは 1 つだけです。このサウンド ミックスのステップでは、サウンドのフォーマット変換も行うので、Bink ファイル内のすべての周波数およびフォーマットのオーディオを使用できます。ここでは、PSP ハードウェア フォーマットである 44,100 Hz、16 ビット、ステレオ フォーマットに自動的に変換します。

当然この変換とミックスは貴重な CPU を消費するので、通常はサウンドをあらかじめミックスしておき、常に 44.1 kHz オーディオ フォーマットを使用することをお勧めします。

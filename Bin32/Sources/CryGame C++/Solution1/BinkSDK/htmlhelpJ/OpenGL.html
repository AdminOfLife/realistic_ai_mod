Bink SDK 1.9c

OpenGL

説明

OpenGL は、3D グラフィック カードを制御するために使用される業界標準の 3D グラフィック API です。

Windows での 3D ビデオ テクスチャ再生のサンプルについては、Exam3D.c ファイルおよび GL サポート ファイル (GLRAD3D.C) を参照してください。また、OpenGL を使用して、高速アルファ平面に基づく透過ビデオ再生を行うこともできます (例は ExamAlph.C と GLRAD3D.C を参照)。Windows OpenGL サンプル コードは、固定機能パイプラインだけを使用します。GL ピクセル シェーダを使用するには (3D ビデオを表示する最善の方法)、独自の GLSL コードを作成するか、サンプル コード (BinkTextures.cpp) と共に Direct3D を使用します。

ソニー プレイステーション 3 では、cG ピクセル シェーダと共に OpenGL を使用して、ビデオ フレームを表示します。これが最も高速で高品質なオプションです。

OpenGL の詳細については、Web サイト GLSetup を参照してください。

Windows、ソニー PS3、Wii、ゲームキューブ、Xbox、および Xbox 360 でピクセル シェーダ ブリッティングを実行するためのクロス プラットフォーム API が用意されています。詳細については、BinkTextures.h ファイルを参照してください。


重要事項 3D システムで非アルファ平面ビデオを再生する場合は、アルファ ブレンディングのレンダリング状態をオフにする必要があります。Bink は、非アルファ ピクセル フォーマット (BINKSURFACE32 など) でアルファ値を設定しません。設定すると、ブリッティング速度が遅くなるためです。非アルファ ビデオを再生しようとして、画面が黒くなる場合は、アルファのレンダリング状態がオンになっている可能性があります。

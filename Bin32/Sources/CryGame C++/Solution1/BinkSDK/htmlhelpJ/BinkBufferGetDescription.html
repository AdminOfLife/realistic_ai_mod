Bink SDK 1.9c

BinkBufferGetDescription

関数 BinkBufferGetDescription は、BinkBuffer が使用しているブリッティング スタイルを表す文字列を返します。

CHAR_PTR BinkBufferGetDescription(
  HBINKBUFFER bink_buf
);

入力パラメータ

bink_buf
BinkBuffer ハンドルを指定します。

戻り値

記述文字列を返します。

説明

この関数は、指定された BinkBuffer が使用するブリッティング スタイルを表す文字列定数を返します。BinkBuffer が使用する方法のタイプを表示するとき (特に BinkBuffer を BINKBUFFERAUTO ブリッティング スタイルで開いた場合) に役立ちます。


グループ: BinkBuffer API
関連する基本データ型: BINKBUFFER_OPEN_FLAGS, CHAR_PTR, HBINKBUFFER

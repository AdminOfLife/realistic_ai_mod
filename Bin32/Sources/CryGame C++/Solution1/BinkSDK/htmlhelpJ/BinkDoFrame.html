Bink SDK 1.9c

BinkDoFrame

BinkDoFrame 関数は、ビデオの次のフレームを解凍します。

S32 BinkDoFrame(
  HBINK bink
);

入力パラメータ

bink
Bink ハンドルを指定します。

戻り値

フレームがスキップされた場合は 1、スキップされなかった場合は 0 が返されます。

説明

この関数は、次の Bink フレームを実際に解凍します。Bink は圧縮された Bink フレームを内部の 3 つの YUV メモリ バッファに解凍するので、BinkCopyToBuffer 関数でこのデータを内部バッファからコピーする必要があります。

オーディオがビデオより遅れると、この関数はビデオ フレームの解凍をスキップします。スキップされたフレームの数を知るには、Bink の要約情報の構造体 (詳細については BinkGetSummary 関数を参照) の SkippedFrames フィールドを確認します。BinkOpen で BINKNOSKIP フラグを指定すると、BinkDoFrame はフレームをスキップしません。

この関数を再び呼び出す前に、BinkNextFrame を呼び出す必要があります。

フレームがスキップされた場合は 1、スキップされなかった場合は 0 が返されます。BinkOpen で BINKNOSKIP を指定した場合は、フレームがスキップされたときに 0 以外の値が返されます。

Bink SDK 1.9c

RADARAMFREE

RADARAMFREE 関数は、すでに RADARAMALLOC 関数を呼び出すことで割り当てられているメモリを解放する関数をポイントします。

void RADLINK RADARAMFREE(
  VOID_PTR ptr
);

入力パラメータ

ptr
RADARAMALLOC が直前に割り当てたポインタを指定します。

説明

この関数は、ニンテンドー ゲームキューブまたは Wii で AX を使用している場合に、オーディオ メモリを解放するために Bink が呼び出す関数を置き換えます。RADARAMALLOC が返すすべてのポインタを解放できる必要があります。


グループ: Bink のオーディオ システムのインターフェイス関数
関連する関数: BinkSoundUseAX, RADARAMALLOC
関連する基本データ型: VOID_PTR, void
関連する構造体: RADARAMCALLBACKS

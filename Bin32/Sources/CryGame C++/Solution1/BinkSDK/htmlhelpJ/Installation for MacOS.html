Bink SDK 1.9c

MacOS へのインストール

説明

PowerPC MacOS 用 Bink SDK は、MacBinary 形式の圧縮アーカイブで提供されます。これをインストールするには、当社の FTP サイトからダウンロードした BinkSDK.sit.bin ファイルをダブルクリックします。

Intel MacOS 用 Bink SDK は、DMG ボリューム イメージとして提供されます。これをマウントするには、BinkMacIntelSDK.dmg ファイルをダブルクリックし、ボリューム内のファイルをマシンの Bink フォルダにドラッグします。

インストールが完了したら、Bink SDK フォルダにある Bink SDK Help のエイリアスをクリックして、オンラインでアクセスできます。

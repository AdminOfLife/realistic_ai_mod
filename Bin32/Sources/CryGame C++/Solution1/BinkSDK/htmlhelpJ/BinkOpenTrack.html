Bink SDK 1.9c

BinkOpenTrack

BinkOpenTrack 関数は、低レベルのオーディオ トラック デコーダのハンドルを開きます。

HBINKTRACK BinkOpenTrack(
  HBINK bink,
  U32 track_index
);

入力パラメータ

bink
Bink ハンドルを指定します。
track_index
開くトラック インデックスを指定します。

戻り値

オーディオの抽出元になるトラック ID のハンドルを返します。

説明

この関数を使用すると、低レベルのサウンド データの抽出元として使用可能なオーディオ トラックを開くことができます。

通常は、アプリケーションがサウンドをアプリケーション固有のオーディオ再生システムに関連付ける場合に使用します。

返される HBINKTRACK ハンドルは、オーディオ トラックのフォーマット情報を含む BINKTRACK 構造体へのポインタでもあります。

Bink ファイル内の最大のトラック数は、BINK 構造体の NumTracks フィールドで指定されます。したがって、NumTracks が 2 の場合、有効なトラック インデックスは 0 か 1 です。

この関数は、ほとんど使用されません。通常は、オーディオの再生は Bink によって完全に自動的に行われます。

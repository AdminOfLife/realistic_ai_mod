Bink SDK 1.9c

基本データ型

説明

Bink は、できるかぎりクロス プラットフォームになるように設計されています。この目的のために、Bink はプラットフォームごとに構成された一連の基本データ型を使用することによって API での一貫性を保証しています。

これらの基本データ型には、整数型、浮動小数点型などがあります。一般的に、アプリケーションと Bink とのインターフェイス部分では、アプリケーション内でも Bink のデータ型を使用することをお勧めします。これは、アプリケーションを他の Bink プラットフォームに移植する作業が迅速で簡単になるためです。

基本データ型
S8 符号付き 8 ビット整数を表します。
U8 符号なし 8 ビット整数を表します。
S16 符号付き 16 ビット整数を表します。
U16 符号なし 16 ビット整数を表します。
S64 符号付き 64 ビット整数を表します。
U64 符号なし 64 ビット整数を表します。
S32 符号付き 32 ビット整数を表します。
U32 符号なし 32 ビット整数を表します。
s32_ptr 符号付き 32 ビット整数へのポインタを表します。
u32_ptr 符号なし 32 ビット整数へのポインタを表します。
F32 標準の 32 ビット浮動小数点型の数値を表します。
F64 標準の 64 ビット倍精度浮動小数点型の数値を表します。
void 標準の C/C++ の void 戻り型を表します。
NULL 標準の C/C++ の NULL ポインタを表します。
OS_WND Win32 上の標準の HWND ウィンドウ ハンドル、または MacOS での標準の WindowPtr を表します。
HCURSOR Win32 の HCURSOR カーソル ハンドルを表します。
LPDIRECTSOUND DirectSound オブジェクトへのポインタを表します。
LPDIRECTDRAW DirectDraw オブジェクトへのポインタを表します。
LPDIRECTDRAWSURFACE DirectDraw サーフェス オブジェクトへのポインタを表します。
LPDIRECT3DSURFACE8 DirectX 8 Direct3D サーフェス オブジェクトへのポインタを表します。
LPDIRECT3DSURFACE9 DirectX 9 Direct3D サーフェス オブジェクトへのポインタを表します。
FSSpec_PTR MacOS の filespec へのポインタを表します。
FSFileID_PTR ニンテンドー DS FSFileID へのポインタを表します。
GDHandle 標準の MacOS のグラフィック デバイスのポインタを表します。
VOID_PTR 標準の C/C++ の untyped メモリ ポインタを表します。
CHAR_PTR 0 で終わる通常の C 形式の文字列を表します。
HBINK 開いている Bink ファイルのハンドルです。
HBINKTRACK 開いている Bink のオーディオ トラックのハンドルです。
HBINKBUFFER 開いている BinkBuffer のハンドルです。
BINKRECT 長方形の領域を表す構造体です。
BINKRECT_ARRAY 長方形の領域を表す構造体の配列です。

Bink SDK 1.9c

BinkGetKeyFrame

BinkGetKeyFrame 関数は、指定されたフレームの付近にあるキー フレームを検索します。

U32 BinkGetKeyFrame(
  HBINK bink,
  U32 frame_num,
  BINK_GETKEY_FLAGS getkey_flags
);

入力パラメータ

bink
Bink ハンドルを指定します。
frame_num
検索するフレーム番号を指定します。
getkey_flags
実行する検索の種類を指定します。

戻り値

最も近いキー フレームを返します。

説明

この関数を使用すると、指定したフレーム番号に最も近いキー フレームを検索できます。任意のフレーム番号を渡すだけで、Bink はフラグのパラメータで指定された検索方法で最も近いキー フレームを検索します。


グループ: Bink API
関連する関数: BinkGoto
関連する基本データ型: BINK_GETKEY_FLAGS, BINK_GOTO_FLAGS, HBINK, U32

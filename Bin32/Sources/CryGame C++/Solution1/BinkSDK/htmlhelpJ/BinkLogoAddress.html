Bink SDK 1.9c

BinkLogoAddress

BinkLogoAddress 関数は、埋め込まれた Bink ロゴ ファイルへのポインタを返します。

VOID_PTR BinkLogoAddress();

戻り値

Bink のロゴを表示する Bink ファイルへのポインタを返します。

説明

この関数は、埋め込みの Bink ロゴ ファイルのアドレスを返します。この関数が返すアドレスは、BINKFROMMEMORY フラグを使用して直接 BinkOpen に渡すことができます。これは、クレジットを表示するために Bink のロゴにアクセスするときに便利です。


グループ: Bink API
関連する関数: BinkOpen
関連する基本データ型: BINK_OPEN_FLAGS, VOID_PTR

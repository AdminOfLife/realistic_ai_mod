Bink SDK 1.9c

GNU/Linux へのインストール

説明

Bink SDK は、当社の FTP サイトからダウンロードされる zip 圧縮ファイルとして提供されます。このファイルを使用するには、ディレクトリに解凍するだけです。

インストールが完了したら、html フォルダにあるファイル index.html をクリックして、オンラインでアクセスできます。このファイルへのリンクを作成することもできます。

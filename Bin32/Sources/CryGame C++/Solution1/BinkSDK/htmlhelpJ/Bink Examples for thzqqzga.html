Bink SDK 1.9c

ソニー PS3 用の Bink サンプル

説明

Bink SDK には、PS3 用のサンプル プログラムがいくつか用意されています。これらのサンプルをビルドするには、新規プロジェクトを作成し、適切な C のソース ファイル (ExamPS3.CPP など) をインクルードし、次に Sony PS3 版 Bink のスタティック ライブラリをプロジェクト (BinkPS3.a) に追加するだけです。

練習用のサンプル ファイルは次のとおりです。

OpenGL フラグメント シェーダを使用して Bink ビデオを表示するサンプルについては、ファイル ExamPS3.CPP を参照してください。このサンプルでは OpenGL ラッパーを使用するので、必ずファイル PSGLTextures.CPP もプロジェクトに追加してください。

Bink のアルファ平面で OpenGL フラグメント シェーダを使用するサンプルについては、ExamPS3A.CPP ファイルを参照してください。このサンプルでは OpenGL ラッパーを使用するので、必ずファイル PSGLTextures.CPP もプロジェクトに追加してください。

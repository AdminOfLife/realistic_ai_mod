Bink SDK 1.9c

BINK

BINK 構造体は、BinkOpen から返される構造体です。

struct BINK
{
  U32 Width;
  U32 Height;
  U32 Frames;
  U32 FrameNum;
  U32 FrameRate;
  U32 FrameRateDiv;
  U32 ReadError;
  BINK_OPEN_FLAGS OpenFlags;
  BINKRECT_ARRAY FrameRects;
  U32 NumRects;
  U32 NumTracks;
};

出力メンバ

Width
Bink ファイルの幅を指定します。
Height
Bink ファイルの高さを指定します。
Frames
Bink ファイルのフレームの数を指定します。
FrameNum
表示するフレームを指定します。
FrameRate
FrameRateDiv との組み合わせにより、Bink のフレーム レートを指定します。フレーム レートを (bink->FrameRate/bink->FrameRateDiv) によって計算します。たとえば、15 fps では、FrameRate は 15 で FrameRateDiv は 1 です。
FrameRateDiv
フレーム レートの除数を指定します。
ReadError
再生中に I/O エラーが発生すると 0 以外に設定されます。
OpenFlags
開いている Bink ファイルで現在使用されているフラグを指定します。
FrameRects
最後に呼び出された BinkGetRects に基づいて、変更された長方形領域を返します。
NumRects
最後に呼び出された BinkGetRects に基づいて、変更された長方形領域の数を返します。
NumTracks
ファイル内のオーディオ トラックの数を返します。

説明

BINK 構造体は、BinkOpen が返す HBINK ハンドルによってポイントされます。ポインタを逆参照することにより、これらの情報フィールドにアクセスできます。

これらの値は、決して変更しないように注意してください。

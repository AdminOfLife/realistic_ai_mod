Bink SDK 1.9c

BINK_BG_CONTROL_FLAGS

基本データ型 BINK_BG_CONTROL_FLAGS は、切り替え先の I/O 状態を指定します。

基本的な値

 値  説明
 BINKBGIOSUSPEND すべてのバックグラウンド I/O を一時停止します。内部 I/O バッファが空の場合、Bink はフォアグラウンドで I/O を実行します (BinkNextFrame 内)。
 BINKBGIORESUME バックグラウンド I/O を再開します。
 BINKBGIOWAIT このフラグと他のフラグを OR 演算して、I/O が一時停止または再開されるまで待機できます。ビデオがすでに再開されている場合に、このフラグと BINKBGIORESUME を使用すると、I/O が即座に実行されます。これは、フォアグラウンドで I/O を強制する場合に便利です。

説明

これらのフラグは、Bink のバックグラウンド I/O システムを制御するために BinkControlBackgroundIO 関数で使用されます。


グループ: Bink API
関連する関数: BinkControlBackgroundIO, BinkNextFrame

Bink SDK 1.9c

BinkBufferSetResolution

関数 BinkBufferSetResolution は、BinkBuffer が開かれるときにビデオの解像度を切り替えるように Bink に指示します。

void BinkBufferSetResolution(
  U32 width,
  U32 height,
  U32 color_depth
);

入力パラメータ

width
切り替え後の画面の幅 (ピクセル) を指定します。
height
切り替え後の画面の高さ (ピクセル) を指定します。
color_depth
切り替え後の画面の色深度を指定します (8、16、24、または 32 ビット)。色深度マクロを使用することもできます。その場合、1 はハイカラー以上 (16 ビット、32 ビット、24 ビットの順に試行)、2 はフルカラー以上 (32 ビット、24 ビット、16 ビットの順に試行)、3 はフルカラーのみ (32 ビット、24 ビットの順に試行) を表します。

説明

この関数は、Bink ファイルの再生に使用する解像度を通知します。この関数は実際に解像度を切り替えることはなく、BinkBuffer を開くときに、ここで指定した解像度が使用されることを要求するだけです。

この関数を使用するには、まずこの関数を呼び出してから、BINKBUFFERRESOLUTION フラグを使用して BinkOpen を呼び出します。

ただし、すでに DirectDraw を使用して解像度を切り替えていた場合は、BinkBuffer API で再び呼び出す必要はありません。

BINKBUFFERDIBSECTION フラグを使用してこの BinkBuffer を開いた場合、Bink は ChangeDisplaySettings を使用してビデオ モードを切り替えます。


グループ: BinkBuffer API
関連する関数: BinkBufferOpen, BinkOpen
関連する基本データ型: BINKBUFFER_OPEN_FLAGS, U32, void

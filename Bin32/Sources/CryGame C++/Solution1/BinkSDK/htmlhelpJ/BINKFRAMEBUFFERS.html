Bink SDK 1.9c

BINKFRAMEBUFFERS

BINKFRAMEBUFFERS 構造体は、BinkGetFrameBuffersInfo の出力および BinkRegisterFrameBuffers への入力として使用される構造体です。

struct BINKFRAMEBUFFERS
{
  S32 TotalFrames;
  U32 YABufferWidth;
  U32 YABufferHeight;
  U32 cRcBBufferWidth;
  U32 cRcBBufferHeight;
  U32 FrameNum;
  BINKFRAMEPLANESET Frames;
};

出力メンバ

TotalFrames
ムービーによって使用される平面セットの数です (すべてのキー フレーム使用するムービーでは 1、その他のムービーでは 2)。
YABufferWidth
Y 平面およびアルファ平面の幅を指定します。
YABufferHeight
Y 平面およびアルファ平面の高さを指定します。
cRcBBufferWidth
cR 平面および cB 平面の幅を指定します。
cRcBBufferHeight
cR 平面および cB 平面の高さを指定します。
FrameNum
現在のアクティブな平面セットを指定します。
Frames
平面セットの配列です (長さは TotalFrames)。

説明

この構造体は、内部 Bink フレーム バッファ セットによって使用されるメモリを管理するために使用します。Bink は、ビデオ フレームごとに 3 または 4 つ (アルファを含む) の平面を使用して、2 つのビデオ フレームを内部的にバッファリングします。この構造体を使用する場合は、アプリケーションで 2 回使用します。最初は、BinkGetFrameBuffersInfo を使用して内部で構造体を入力する場合、もう 1 回は、メモリ ポインタを割り当てた後に BinkRegisterFrameBuffers で使用されます。


グループ: Bink API
関連する関数: BinkGetFrameBuffersInfo, BinkRegisterFrameBuffers
関連する基本データ型: S32, U32
関連する構造体: BINKFRAMEPLANESET, BINKPLANE

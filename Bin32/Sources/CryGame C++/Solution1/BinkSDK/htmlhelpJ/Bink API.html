Bink SDK 1.9c

Bink API

説明

Bink API は、Bink ファイルの各フレームを解凍するときに使用される API です。ファイルのオープン、ビデオ フレームの解凍、次のフレームの待機、次のフレームへの移動などに使用します。

しかし、画面には何も表示しません。Bink API は、単にリニアなメモリ バッファにデータを配置するだけです。このバッファには、DirectDraw サーフェス、DIBSection、または単に malloc で割り当てた通常のメモリを使用できます。データの配置先が Bink API 側で考慮されることはありません。

Bink は直接画面にブリットしないので、柔軟性が高くなっています。Bink を使用すると、ビデオを圧縮するだけでなく、ビットマップ、テクスチャ、ライト マップ、ハイト マップなどを圧縮できます。任意の長方形のデータ セットを圧縮できます。

ただし、ブリッティングはほとんどのプラットフォームで (特に Win32 では) 複雑な処理であるため、Win32 と MacOS では既製のブリッティング API (BinkBuffer API) を用意しています。この API は Bink API からは独立しているため、使用するかどうかはユーザーの判断に任されています。BinkBuffer API については、この後のマニュアルで説明します。

Bink API
BinkOpen 指定された Bink ファイルを開いて解凍します。
BinkMacOpen 指定された Bink ファイルを開いて解凍します。
BinkNDSOpen 指定された Bink ファイルを開いて解凍します。
BINK BinkOpen から返される構造体です。
BINK_OPEN_FLAGS Bink ファイルを開く特殊な方法を表します。
BinkClose 指定された Bink ハンドルを閉じます。
BinkDoFrame ビデオの次のフレームを解凍します。
BinkCopyToBuffer Bink 内のメモリ バッファから指定されたメモリ アドレスにビデオのフレームをコピーします。
BinkCopyToBufferRect Bink 内のメモリ バッファから指定されたメモリ アドレスにビデオのフレームの一部をコピーします。
BINK_COPY_FLAGS Bink 内の YUV データの特殊な変換方法や変換先のバッファのタイプを表します。
BinkNextFrame Bink ファイルのビデオの次のフレームに移動します。
BinkWait 次の Bink フレームに移動する時間になったことを (ファイルのフレーム レートに基づいて) 通知します。
BinkPause Bink ファイルの再生を一時停止および再開します。
BinkGetRects 最後に解凍したビデオ フレームの変更された長方形領域を返します。
BINK_GETRECTS_FLAGS BinkGetRects 関数で使用する長方形の一覧の型を指定します。
BinkGoto Bink ファイル内の新しいフレームに移動します。
BINK_GOTO_FLAGS 実行する移動の種類を指定します。
BinkGetFrameBuffersInfo Bink が Y 平面、cR 平面、および cB 平面を保存する内部フレーム バッファに関する詳細を返します。
BINKFRAMEBUFFERS BinkGetFrameBuffersInfo の出力および BinkRegisterFrameBuffers への入力として使用される構造体です。
BINKFRAMEPLANESET 平面のセット (Y、cR、cB、およびアルファに対して 1 つ) を記述する構造体です。
BINKPLANE 1 つの 2D 平面を記述する構造体です。
BinkRegisterFrameBuffers Y、cR、および cB 平面を解凍するために Bink が使用するフレーム バッファ ポインタを設定します。
BinkGetKeyFrame 指定されたフレームの付近にあるキー フレームを検索します。
BINK_GETKEY_FLAGS 実行する検索の種類を指定します。
BinkShouldSkip フレームまたはブリットをスキップする必要があるほどビデオがオーディオより遅れているかどうかを報告します。
BinkService Bink がサウンド システムにサービスを提供するための明示的なアイドル時間を提供します。
BinkSetVolume Bink のサウンド トラックの音量を設定します。
BinkSetPan Bink のサウンド トラックのスピーカーのパン (右対左のバランス) を設定します。
BinkSetMixBins Xbox または Xbox 360 のサウンド トラックのミックス ビンの出力先を設定します。
BinkSetMixBinVolumes Xbox、Xbox 360、ソニー PS3、または Miles Sound System の個々のミックス ビンの音量を設定します。
BinkSetVideoOnOff BinkDoFrame を呼び出したときにビデオ フレームを解凍しないように Bink に指示します。
BinkSetSoundOnOff オーディオの解凍を禁止し、オーディオのリソースを解放するように Bink に指示します。
BinkGetSummary 指定された Bink ハンドルに関する再生情報を返します。
BINKSUMMARY 再生情報の要約を BinkGetSummary から返すために使用する構造体です。
BinkGetRealtime 指定された Bink ハンドルに関するリアルタイムの再生情報を返します。
BINKREALTIME リアルタイムの再生情報を BinkGetRealtime から返すために使用する構造体です。
BinkControlBackgroundIO Bink のバックグラウンド I/O システムを一時停止または再開します。
BINK_BG_CONTROL_FLAGS 切り替え先の I/O 状態を指定します。
BinkGetPalette BINKSURFACEP8 サーフェス タイプのグローバル Bink パレットを返します。
BinkGetTrackID 指定したトラック インデックスのトラック ID を返します。
BinkOpenTrack 低レベルのオーディオ トラック デコーダのハンドルを開きます。
BINKTRACK BinkOpenTrack から返される構造体です。
BinkCloseTrack 低レベルのオーディオ トラック デコーダのハンドルを閉じます。
BinkGetTrackData 現在のビデオ フレームからすべてのサウンドを抽出します。
BinkSetFrameRate Bink ファイルに組み込まれたフレーム レートを上書きします。
BinkSetSoundTrack Bink ファイルのデフォルトのサウンド トラックを上書きします。
BinkSetSimulate 指定されたデバイスへの転送速度をシミュレートするように Bink に指示します。
BinkSetIOSize Bink が読み込む I/O メモリのバッファ サイズを指定します。
BinkSetIO 置き換えるファイル I/O 関数のセットを指定します。
BinkSetMemory 置き換えるメモリ管理関数のセットを指定します。
RADMEMALLOC Bink にメモリを割り当てる関数をポイントします。
RADMEMFREE すでに RADMEMALLOC 関数を呼び出すことで割り当てられているメモリを解放する関数をポイントします。
BinkGetError 最新のグローバル Bink エラーを説明する文字列を返します。
BinkLogoAddress 埋め込まれた Bink ロゴ ファイルへのポインタを返します。
BinkLoad Xbox 上にある Bink のセクションを (BinkUnload でアンロードされていた場合) リロードします。
BinkUnload Xbox で Bink のセクションをアンロードします。
BinkLoadConverter Bink YUV から RGB への少なくとも 1 つの変換を (BinkUnloadConverter でアンロードされていた場合) リロードします。
BinkUnloadConverter は、Bink YUV から RGB への少なくとも 1 つの変換をアンロードします。
BINK_CONVERTER_FLAGS Bink YUV 変換のクラスをロードするかどうかを指定します。

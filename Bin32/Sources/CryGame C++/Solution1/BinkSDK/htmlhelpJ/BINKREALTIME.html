Bink SDK 1.9c

BINKREALTIME

BINKREALTIME 構造体は、リアルタイムの再生情報を BinkGetRealtime から返すために使用する構造体です。

struct BINKREALTIME
{
  U32 FrameNum;
  U32 FrameRate;
  U32 FrameRateDiv;
  U32 Frames;
  U32 FramesTime;
  U32 FrameVideoDecompTime;
  U32 FrameAudioDecompTime;
  U32 FrameReadTime;
  U32 FrameIdleReadTime;
  U32 FrameThreadReadTime;
  U32 FramesBlitTime;
  U32 ReadBufferSize;
  U32 ReadBufferUsed;
  U32 FramesDataRate;
};

出力メンバ

FrameNum
現在のフレームの番号を指定します。
FrameRate
FrameRateDiv との組み合わせにより、Bink のフレーム レートを指定します。フレーム レートを (realtime->FrameRate/realtime->FrameRateDiv) によって計算します。たとえば、15 fps では、FrameRate は 15 で FrameRateDiv は 1 です。
FrameRateDiv
フレーム レートの除数を指定します。
Frames
フレームのウィンドウのフレームの数を指定します。
FramesTime
フレーム ウィンドウに表示する範囲の合計時間 (ミリ秒) を指定します。
FrameVideoDecompTime
フレーム ウィンドウへのビデオの解凍にかかる時間を指定します。
FrameAudioDecompTime
フレーム ウィンドウへのオーディオの解凍にかかる時間を指定します。
FrameReadTime
フォアグラウンドでデバイスからフレーム ウィンドウへの読み込みにかかる時間を指定します。
FrameIdleReadTime
アイドル時間の間にデバイスからフレーム ウィンドウへの読み込みにかかる時間を指定します。
FrameThreadReadTime
バックグラウンド スレッドでデバイスからフレーム ウィンドウへの読み込みにかかる時間を指定します。
FramesBlitTime
フレーム ウィンドウへの画面のブリッティングにかかる時間を指定します。
ReadBufferSize
現在の最大の I/O バッファのサイズをバイト数で指定します。
ReadBufferUsed
先読みに使用される現在の I/O バッファのサイズを指定します (この値を ReadBufferSize で割って使用率をパーセントで表します)。
FramesDataRate
現在のフレーム ウィンドウのデータ レートを 1 秒あたりのバイト数で指定します。

説明

BINKREALTIME 構造体は、BinkGetRealtime 関数で使用します。再生中の Bink ファイルの再生に関するリアルタイムの統計を返します。


グループ: Bink API
関連する関数: BinkGetRealtime
関連する基本データ型: U32

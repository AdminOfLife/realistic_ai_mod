RAD_IOPSetSPUVoiceOnOff

関数 RAD_IOPSetSPUVoiceOnOff は、1 つ以上の SPU 音声を開始または停止します。

S32 RAD_IOPSetSPUVoiceOnOff(
  U32 core0_voice_mask,
  U32 core1_voice_mask,
  S32 onoff
);

入力パラメータ

core0_voice_mask
開始または停止するコア 0 の音声のビット マスクを指定します。
core1_voice_mask
開始または停止するコア 1 の音声のビット マスクを指定します。
onoff
指定した音声を開始する場合は 1、停止する場合は 0 を使用します。

戻り値

正常ならば 0 以外の値、失敗の場合は 0 を返します。

説明

この関数は、1 つ以上の SPU 音声の再生を開始または停止するように SPU に指示します。

実用上は、この関数が音声を開始および停止すると考えてかまいませんが、ハードウェア レベルで行われている動作は、厳密にはそうではありません。

実際は、音声は常にデータをデコードしているため、SPU ハードウェアで音声が「開始」または「停止」されるという言い方は少し変です。正確には、音声を開始する場合、RAD_IOPSetSPUVoiceStartAddress を使用して設定された開始アドレスにジャンプし、RAD_IOPSetSPUVoiceADSR によって指定された音量エンベロープを開始するようにハードウェアに指示していると言うほうが適切です。音声を停止する場合、実際には、ハードウェアにエンベロープのリリース部分を開始するように指示しているだけです (通常はオーディオ音量をゼロに下げるだけです。これで、再生が停止したように見えます)。

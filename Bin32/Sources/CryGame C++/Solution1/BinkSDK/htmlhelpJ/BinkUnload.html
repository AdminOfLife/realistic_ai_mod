Bink SDK 1.9c

BinkUnload

BinkUnload 関数は、Xbox で Bink のセクションをアンロードします。

void BinkUnload();

説明

この関数は、Bink をメモリからアンロードします。これによってメモリの消費が約 100K 少なくなります。この関数を呼び出して Bink のすべてのメモリを解放する前に、Bink の YUV 変換をアンロードする必要があります。たとえば、メモリから Bink を完全にアンロードするには、次のようにします。

// Bink をアンロード
BinkUnloadConverters( BINKCONVERTERSALL );
BinkUnload();

// 次に Bink をリロード
BinkLoad();

// さらにゲームで使用する変換をリロード
BinkLoadConverters( BINKSURFACE32 );

一度 Bink をアンロードすると、BinkLoad 以外の Bink のすべての関数は、呼び出す前に Bink をリロードする必要があります。


グループ: Bink API
関連項目: Xbox のメモリ管理
関連する関数: BinkLoad, BinkUnloadConverter

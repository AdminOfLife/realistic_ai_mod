Bink SDK 1.9c

BINKRECT

BINKRECT 構造体は、長方形の領域を表す構造体です。

struct BINKRECT
{
  S32 Left;
  S32 Top;
  S32 Width;
  S32 Height;
};

出力メンバ

Left
左座標を指定します (基点は 0)。
Top
上座標を指定します (基点は 0)。
Width
長方形の幅を指定します (基点は 1)。
Height
長方形の高さを指定します (基点は 1)。


グループ: 基本データ型
関連する基本データ型: S32

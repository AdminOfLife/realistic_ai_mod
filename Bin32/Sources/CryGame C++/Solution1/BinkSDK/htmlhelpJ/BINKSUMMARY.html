Bink SDK 1.9c

BINKSUMMARY

BINKSUMMARY 構造体は、再生情報の要約を BinkGetSummary から返すために使用する構造体です。

struct BINKSUMMARY
{
  U32 Width;
  U32 Height;
  U32 TotalTime;
  U32 FileFrameRate;
  U32 FileFrameRateDiv;
  U32 FrameRate;
  U32 FrameRateDiv;
  U32 TotalOpenTime;
  U32 TotalFrames;
  U32 TotalPlayedFrames;
  U32 SkippedFrames;
  U32 SkippedBlits;
  U32 SoundSkips;
  U32 TotalBlitTime;
  U32 TotalReadTime;
  U32 TotalVideoDecompTime;
  U32 TotalAudioDecompTime;
  U32 TotalIdleReadTime;
  U32 TotalBackReadTime;
  U32 TotalReadSpeed;
  U32 SlowestFrameTime;
  U32 Slowest2FrameTime;
  U32 SlowestFrameNum;
  U32 Slowest2FrameNum;
  U32 AverageDataRate;
  U32 AverageFrameSize;
  U32 HighestMemAmount;
  U32 TotalIOMemory;
  U32 HighestIOUsed;
  U32 Highest1SecRate;
  U32 Highest1SecFrame;
};

出力メンバ

Width
Bink ファイルの幅を指定します。
Height
Bink ファイルの高さを指定します。
TotalTime
再生時間の合計 (ミリ秒) を指定します。
FileFrameRate
FileFrameRateDiv との組み合わせにより、Bink のフレーム レートの初期値を指定します。フレーム レートを (summary->FileFrameRate/summary->FileFrameRateDiv) によって計算します。ファイルのフレーム レートとは、そのファイルのフレーム レートを BinkSetFrameRate で変更する前の値です。
FileFrameRateDiv
Bink ファイルのフレーム レートの除数の初期値を指定します。
FrameRate
FrameRateDiv との組み合わせにより、Bink のフレーム レートを指定します。フレーム レートを (summary->FrameRate/summary->FrameRateDiv) によって計算します。
FrameRateDiv
フレーム レートの除数を指定します。
TotalOpenTime
解凍するために Bink ファイルを開いて準備する時間を指定します。
TotalFrames
Bink ファイルのフレームの数を指定します。
TotalPlayedFrames
再生するフレームの数を指定します。
SkippedFrames
再生中にスキップするフレームの数を指定します。
SkippedBlits
再生中にスキップするブリッティング処理の数を指定します。
SoundSkips
再生中にサウンドがドロップした回数を指定します。
TotalBlitTime
ブリッティングにかかかる時間 (ミリ秒) を指定します。
TotalReadTime
フォアグラウンドでの読み込みにかかる時間 (ミリ秒) を指定します。
TotalVideoDecompTime
ビデオの解凍にかかかる時間 (ミリ秒) を指定します。
TotalAudioDecompTime
オーディオの解凍にかかかる時間 (ミリ秒) を指定します。
TotalIdleReadTime
アイドル時間での読み込みにかかる時間 (ミリ秒) を指定します。
TotalBackReadTime
バックグラウンド スレッドでの読み込みにかかる時間 (ミリ秒) を指定します。
TotalReadSpeed
デバイスからの読み込みの転送速度 (バイト/秒) を指定します。
SlowestFrameTime
最も遅い 1 つのフレームの時間 (ミリ秒) を指定します。
Slowest2FrameTime
2 番目に遅い 1 つのフレームの時間 (ミリ秒) を指定します。
SlowestFrameNum
最も遅い 1 つのフレームの番号を指定します。
Slowest2FrameNum
2 番目に遅い 1 つのフレームの番号を指定します。
AverageDataRate
ムービーの平均のデータ レート (バイト/秒) を指定します。
AverageFrameSize
フレームの平均サイズ (バイト数) を指定します。
HighestMemAmount
再生中に 1 回で割り当てられた最大メモリ容量 (バイト数) を指定します。
TotalIOMemory
Bink ファイルで先読みに使用する I/O バッファのサイズ (バイト数) を指定します。
HighestIOUsed
先読み I/O バッファを満たす最高レベル (バイト数) を指定します。
Highest1SecRate
Bink ファイルで最高の 1 秒間のデータ レート (バイト/秒) を指定します。
Highest1SecFrame
Bink ファイル内で、1 秒間のデータ レートが最も高い開始フレームを指定します。

説明

BINKSUMMARY 構造体は、BinkGetSummary 関数で使用します。この関数は Bink ファイル全体の再生に関する統計を返します。


グループ: Bink API
関連する関数: BinkGetSummary, BinkSetFrameRate
関連する基本データ型: U32

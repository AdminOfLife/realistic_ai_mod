RAD_IOPRegisterTimerCallback

関数 RAD_IOPRegisterTimerCallback は、タイマー構造体を登録して、特定の周期でユーザーのタイマー インタラプト コールバックを呼び出します。

S32 RAD_IOPRegisterTimerCallback(
  RAD_IOP_TIMER_DATA *timer_data
);

出力パラメータ

timer_data
コールバック情報を指定します。

戻り値

タイマーが登録された場合は 0 以外の値、失敗した場合は 0 が返されます。

説明

この関数は、タイマー コールバック構造体を登録して、タイマー インタラプト コールバックを呼び出します。タイマーは、実際には RAD_IOPStartTimer の呼び出しまで開始しません。

この構造体は、タイマーが RAD_IOPUnregisterTimerCallback 関数で登録解除されるまで、有効にしておく必要があります。

タイマー コールバック関数は、インタラプトで呼び出されるため、コードは必ずインタラプト セーフにする必要があります。

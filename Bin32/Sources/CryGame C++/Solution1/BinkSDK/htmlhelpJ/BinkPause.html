Bink SDK 1.9c

BinkPause

BinkPause 関数は、Bink ファイルの再生を一時停止および再開します。

S32 BinkPause(
  HBINK bink,
  S32 pause
);

入力パラメータ

bink
Bink ハンドルを指定します。
pause
一時停止するか (0 以外)、ビデオ再生を再開するか (0) を指定します。

戻り値

ビデオが一時停止中か再生中かを返します。

説明

この関数で、Bink の再生を簡単に一時停止できます。0 以外の値を渡してビデオを一時停止し、一時停止した後で 0 を渡して再生を再開します。一度この関数を呼び出すと、再生を再開するまで BinkWait は 1 を返します。この関数を呼び出しすと、オーディオも一時停止します。


グループ: Bink API
関連する関数: BinkWait
関連する基本データ型: HBINK, S32

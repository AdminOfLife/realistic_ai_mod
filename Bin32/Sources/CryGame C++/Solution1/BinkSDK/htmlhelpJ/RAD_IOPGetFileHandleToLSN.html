RAD_IOPGetFileHandleToLSN

関数 RAD_IOPGetFileHandleToLSN は、CD または DVD 上の論理セクタ番号を直接アドレス指定するファイル ハンドルを作成します。

U32 RAD_IOPGetFileHandleToLSN(
  U32 LSN
);

入力パラメータ

LSN
CD または DVD 上の論理セクタ番号を指定します。

戻り値

SPU メモリへのファイル ハンドル。

説明

この関数は、CD または DVD 上の特定のセクタからの非同期の読み取りに使用できるファイル ハンドルを作成します。

この種類のファイル ハンドルを閉じる必要はありません。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPQueueAsyncOpenFile, RAD_IOPQueueAsyncReadAtBottom, RAD_IOPQueueAsyncReadAtTop
関連する基本データ型: U32

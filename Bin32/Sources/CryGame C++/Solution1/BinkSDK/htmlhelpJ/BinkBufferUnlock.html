Bink SDK 1.9c

BinkBufferUnlock

関数 BinkBufferUnlock は、BinkBufferLock によってロックされていた BinkBuffer のロックを解除します。

S32 BinkBufferUnlock(
  HBINKBUFFER bink_buf
);

入力パラメータ

bink_buf
BinkBuffer ハンドルを指定します。

戻り値

BinkBuffer が正常にロック解除されたか (0 以外の値)、されていないか (0) を返します。

説明

この関数は、ロックされている BinkBuffer のロックを解除します。BinkBufferLock 関数でロックしたすべての BinkBuffer は、できるだけ早くロック解除する必要があります。BinkBuffer のロックを解除できない場合は、通常 DirectDraw によって表示がロックされ、再起動が必要になります。

DirectDraw の場合は、DirectDraw ロックをステップ実行することはできません。したがって、BinkBufferLock を単独のステップで呼び出さないでください。呼び出すとマシンがロックします。必ず BinkBufferUnlock 呼び出しの後にブレーク ポイントを設定して、そのブレーク ポイントまで実行します。

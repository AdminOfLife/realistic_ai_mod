Bink SDK 1.9c

BINK_GETRECTS_FLAGS

基本データ型 BINK_GETRECTS_FLAGS は、BinkGetRects 関数で使用する長方形の一覧の型を指定します。

基本的な値

 値  説明
 BINKSURFACEFAST 多数の小さな長方形を返します (高速ブリッタ用)。
 BINKSURFACESLOW 1 つまたは 2 つの長方形を返します (低速ブリッタ用)。
 BINKSURFACEDIRECT 長方形をまったく返しません (たとえば、プライマリ サーフェスにコピーするとき)。

説明

これらのフラグを BinkGetRects 関数で使用して、選択する変更された領域の種類を指定します。

Bink SDK 1.9c

ロックとロック解除

説明

BinkBuffer 構造体の buffer および buffer_pitch フィールドにアクセスするには、その前に BinkBuffer をロックおよびロック解除することが必要です。これは、BinkBuffer が基盤となる DirectDraw 構造体をロックおよびロック解除できるようにするためです。

ただし、DirectDraw は BinkBufferLock/BinkBufferUnlock ブロック内でロックされることがあるため、デバッガで BinkBuffer ブロック内に入ることはできません。この機能を実行する必要がある場合は、BINKBUFFERDIBSECTION フラグを使用して BinkBuffer を開きます (この場合、DirectDraw をロックする必要はありません)。


グループ: 背景にあるコンセプト
関連する関数: BinkBufferLock, BinkBufferOpen, BinkBufferUnlock
関連する基本データ型: BINKBUFFER_OPEN_FLAGS

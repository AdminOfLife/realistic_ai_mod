Bink SDK 1.9c

BinkBufferSetScale

関数 BinkBufferSetScale は、ブリッティング中に使用する新しい縮小または拡大倍率を設定します。

S32 BinkBufferSetScale(
  HBINKBUFFER bink_buf,
  U32 width,
  U32 height
);

入力パラメータ

bink_buf
BinkBuffer ハンドルを指定します。
width
ブリッティングの拡大または縮小後の幅を指定します。
height
ブリッティングの拡大または縮小後の高さを指定します。

戻り値

BinkBuffer が正常にこの拡大縮小率を使用できたか (0 以外の値)、指定された拡大縮小率が無効だったか (0) を返します。

説明

この関数は、新たに使用する縮小/拡大率を設定します。ブリッティング スタイルによっては縮小または拡大を設定できないので、BinkBuffer を開くときには必ず BINKBUFFERSTRETCHX、BINKBUFFERSTRETCHY、BINKBUFFERSHRINKX、または BINKBUFFERSHRINKY 定数のいずれか 1 つを指定する必要があります。指定しないと拡大縮小が無視されることがあります。

整数の係数 (2 倍など) を使用すると、ほとんどの場合 (ハードウェアの場合も含めて)、拡大縮小は高速になります。

この関数が正常に実行されると、BinkBuffer 構造体の WindowWidth と WindowHeight フィールドが新しいサイズに更新されます。

この関数は、BinkBuffer が拡大縮小を実行できれば 0 以外の値、実行できなければ 0 を返します。

RAD_IOPSetSPUVoiceStartAddress

関数 RAD_IOPSetSPUVoiceStartAddress は、1 つ以上の SPU 音声の再生を開始するアドレスを設定します。

S32 RAD_IOPSetSPUVoiceStartAddress(
  U32 core0_voice_mask,
  U32 core1_voice_mask,
  U32 start_address
);

入力パラメータ

core0_voice_mask
設定するコア 0 の音声のビット マスクを指定します。
core1_voice_mask
設定するコア 1 の音声のビット マスクを指定します。
start_address
再生を開始する SPU RAM 上の開始アドレスを指定します。

戻り値

正常ならば 0 以外の値、失敗の場合は 0 を返します。

説明

この関数は、1 つ以上の SPU 音声の再生を開始する位置を SPU に指示します。

通常、RAD_IOP 非同期 IO 呼び出しの 1 つを使用して、VAG ファイルを SPU RAM に読み込み、この関数を使用して、1 つ以上の音声の位置を指定します。


グループ: ソニー PS2 用 RAD_IOP SPU ハードウェア音声 API
関連する関数: RAD_IOPSetSPUVoiceLoopAddress, RAD_IOPSetSPUVoiceOnOff
関連する基本データ型: S32, U32

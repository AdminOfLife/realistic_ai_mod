Win32 での Bink DLL の配布

説明

Bink を使用するアプリケーションを配布する場合は、Bink DLL (BinkW32.DLL) をアプリケーションと同じディレクトリに置きます。

BinkW32.DLL を Windows ディレクトリや Windows のシステム ディレクトリに置かないでください。BinkW32.DLL は、必ずアプリケーション ディレクトリに置いてください。

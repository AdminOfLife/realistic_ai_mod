Bink SDK 1.9c

BinkControlBackgroundIO

BinkControlBackgroundIO 関数は、Bink のバックグラウンド I/O システムを一時停止または再開します。

S32 BinkControlBackgroundIO(
  HBINK bink,
  BINK_BG_CONTROL_FLAGS control
);

入力パラメータ

bink
Bink ハンドルを指定します。
control
バックグラウンドの I/O システムを制御します。

戻り値

一時停止された場合は 1、再開された場合は 0 を返します。

説明

この関数を使用して、Bink のバックグラウンド I/O を制御できます。通常、この関数は、ビデオの再生時にゲーム データをロードするために使用されます。

ビデオの再生時にデータをロードするには、Bink I/O バッファへのデータの設定とゲーム I/O の実行を交互に行う方法が最適です。BinkGetRealtime を使用して Bink バッファの空き容量をモニタし、バッファがいっぱいになったらバックグラウンド I/O を一時停止し、バッファが空になったら再開できます。

再生時にゲーム データをロードするサンプルについては、このセクションを参照してください。

この関数を使用して、ムービーが再開されたら BINKBGIORESUME および BINKBGIOWAIT フラグを使用することでフォアグラウンド I/O を強制できます。

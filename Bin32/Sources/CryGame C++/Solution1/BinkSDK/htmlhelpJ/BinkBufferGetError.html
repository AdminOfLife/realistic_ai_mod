Bink SDK 1.9c

BinkBufferGetError

BinkBufferGetError 関数は、最新のグローバル BinkBuffer エラーを説明する文字列を返します。

CHAR_PTR BinkBufferGetError();

戻り値

最新のグローバル BinkBuffer エラーを返します。

説明

この関数は、最新のグローバル BinkBuffer エラーを説明する文字を返します。これは、BinkBufferOpen が失敗した場合に特に便利です。


グループ: BinkBuffer API
関連する関数: BinkBufferOpen
関連する基本データ型: CHAR_PTR

ニンテンドー Wii プラットフォーム

説明

Bink は、ニンテンドーが誇る Wii コンソールにも対応するようになりました。Wii はバランスのとれた本体に優れた GPU と CPU を備えています。ゲームキューブと同様に、Wii のオペレーティング システムは、小さなサイズと充実した機能の両方を兼ね備えた理想のコンソールと言えます。Bink は、この魔法の小箱が持つ機能を十分に引き出すことができます。

このプラットフォームでは、Bink の使用も統合もきわめて簡単です。必要なことはすべてこの章で説明します。

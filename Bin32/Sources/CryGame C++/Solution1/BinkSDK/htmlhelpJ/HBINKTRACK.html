Bink SDK 1.9c

HBINKTRACK

基本データ型 HBINKTRACK は、開いている Bink のオーディオ トラックのハンドルです。

説明

HBINKTRACK ハンドルは、BinkOpenTrack 関数から返されます。

また、開いている Bink のオーディオ トラックの情報を含む BINKTRACK 構造体をポイントします。

BinkOpenTrack の呼び出しが失敗した場合、このハンドルは 0 になります。


グループ: 基本データ型
関連する関数: BinkCloseTrack, BinkGetTrackData, BinkOpenTrack
関連する構造体: BINKTRACK

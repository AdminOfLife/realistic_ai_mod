Bink SDK 1.9c

RAD_IOPGetIOStatus

関数 RAD_IOPGetIOStatus は、失敗した IO 要求のエラー コードを返します。

S32 RAD_IOPGetIOStatus();

戻り値

エラー コードを返します。これは、再生元のデバイスによって sceCdRead 関数または C ランタイム読み取り関数の結果です。

説明

この関数は、バックグラウンドのファイル読み取りが RAD_IOP_STATUS_ERROR コードで失敗したときにエラー コードを返します。CD または DVD から再生している場合、値は sceCdRead 関数の結果です。他のデバイスでは、結果は C ランタイムの読み取り関数の結果です。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPQueueAsyncReadAtBottom, RAD_IOPQueueAsyncReadAtTop
関連する基本データ型: RAD_IOP_STATUS_VALUES, S32
関連する構造体: RAD_IOP_ASYNC_READ_DATA

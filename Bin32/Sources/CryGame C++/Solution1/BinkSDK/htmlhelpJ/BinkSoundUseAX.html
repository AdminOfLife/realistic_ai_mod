Bink SDK 1.9c

BinkSoundUseAX

BinkSoundUseAX 関数は、ニンテンドー ゲームキューブまたは Wii で AX をサウンド出力システムとして使用するように Bink に指示します。

S32 BinkSoundUseAX(
  RADARAMCALLBACKS const *callbacks
);

入力パラメータ

callbacks
使用する ARAM メモリ管理のコールバックを含む RADARAMCALLBACKS 構造体へのポインタ。

戻り値

Bink の出力レイヤが AX に切り替わったかどうかを返します (正常に終了した場合は 0 以外、失敗した場合は 0)。

説明

この関数 (実際はマクロ) は、ニンテンドー ゲームキューブまたは Wii でのオーディオ出力に AX を使用するように Bink に指示します。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前に呼び出さなければ、この関数は失敗します。


グループ: Bink のオーディオ システムのインターフェイス関数
関連項目: ニンテンドー ゲームキューブのサウンド, ニンテンドー Wii のサウンド
関連する関数: BinkOpen, BinkSetSoundSystem, BinkSoundUseDirectSound, BinkSoundUseLibAudio, BinkSoundUseMiles, BinkSoundUseMusyX, BinkSoundUseNDSSound, BinkSoundUsePSPSound, BinkSoundUseRAD_IOP, BinkSoundUseSDLMixer, BinkSoundUseSoundManager, BinkSoundUseWaveOut, RADARAMALLOC, RADARAMFREE
関連する基本データ型: S32
関連する構造体: RADARAMCALLBACKS

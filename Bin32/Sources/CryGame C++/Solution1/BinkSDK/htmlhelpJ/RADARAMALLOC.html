Bink SDK 1.9c

RADARAMALLOC

RADARAMALLOC 関数は、AX をニンテンドー ゲームキューブまたは Wii で使用する場合に、Bink の代わりにオーディオ メモリを割り当てる関数をポイントします。

VOID_PTR RADLINK RADARAMALLOC(
  U32 num_bytes
);

入力パラメータ

num_bytes
割り当てるメモリをバイト数で指定します。

戻り値

新しいポインタまたは 0 を返します。

説明

この関数は、ニンテンドー ゲームキューブまたは Wii で AX を使用している場合に、オーディオ メモリを割り当てるために Bink が呼び出す関数を置き換えます。num_bytes バイト以上のオーディオ メモリへのポインタを返す必要があります。

実際には Wii に別のオーディオ RAM はありませんが、MEM2 からオーディオ メモリを割り当てる場合のために、別のメモリ割り当て関数を残してあります。

割り当てに失敗した場合、0 を返すと Bink は割り当てに失敗します。

Bink SDK 1.9c

Xbox 用の Bink サンプル

説明

Bink SDK には、Xbox 用のサンプル プログラムがいくつか用意されています。Visual C でこれらのサンプルを作成するには、新規プロジェクトを作成して適切な C のソース ファイル (ExamXbox.C など) をインクルードし、次に Xbox 用の Bink のスタティック ライブラリをプロジェクト (BinkXbox.LIB または BinkXbox.LTCG.LIB) に追加するだけです。

練習用のサンプル ファイルは次のとおりです。

オーバーレイを使用して Bink ビデオを表示するサンプルについては、ファイル ExamXbox.C を参照してください。ゲーム内で簡単にビデオを表示するには、この方法が最適です。

YUV Direct3D テクスチャを使用して Bink ビデオを表示するサンプルについては、ファイル ExamXTex.C を参照してください。このサンプルでは Direct3D ラッパーを使用するので、必ずファイル XBXRAD3D.CPP もプロジェクトに追加してください。

Direct3D で Bink のアルファ平面を使用するサンプルについては、ファイル ExamXAlf.C を参照してください。このサンプルでは Direct3D ラッパーを使用するので、必ずファイル XBXRAD3D.CPP もプロジェクトに追加してください。

Bink で Direct3D バック バッファにコピーするサンプルについては、ファイル ExamXbb.C を参照してください。ただし、オーバーレイを使用する方法がほとんど常に最善の方法です。

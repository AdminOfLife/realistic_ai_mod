Bink SDK 1.9c

BinkSetSoundTrack

BinkSetSoundTrack 関数は、Bink ファイルのデフォルトのサウンド トラックを上書きします。

void BinkSetSoundTrack(
  U32 track_count,
  u32_ptr track_id
);

入力パラメータ

track_count
再生するトラックの数 (track_ids 配列にあるトラックの数) を指定します。サウンドをまったく再生しない場合は、このパラメータを 0 にします。
track_id
再生するサウンド トラック ID の配列をポイントします。

説明

この関数を使用して、Bink ファイルを開いたときに再生するサウンド トラック ID を設定できます。デフォルトでは、Bink はファイルの最初のサウンド トラックを再生します。

どのサウンド トラックも再生しない場合は、パラメータ track_count を 0 にします。

この関数を使用するには、最初にこの関数を呼び出して、次に BINKSNDTRACK フラグを BinkOpen の呼び出しに追加します。

この関数を使用して、Xbox、Xbox 360、ソニー PS3、または Miles Sound System で 5.1 チャンネル オーディオを再生できます。


グループ: Bink API
関連項目: Bink のオーディオ トラック, ソニー PS3 のサウンド, Xbox のサウンド, Xbox 360 のサウンド
関連する関数: BinkOpen, BinkSetSoundOnOff
関連する基本データ型: BINK_OPEN_FLAGS, U32, u32_ptr, void

Bink SDK へようこそ。お買い求めくださり、ありがとうございます。

本ドキュメントでは、すべてのプラットフォームの Bink SDK について説明します。本ドキュメントにはかなりの労力が費やされています。使いこなせるようになるために役立てば幸いです。参考になったこと、参考にならなかったことについてお知らせください。Bink API 本体と同様に、ドキュメントの改善にも努めていきます。それにはお客様のご意見が大切です。

また、このマニュアルも常に改訂されることにご注意ください。Bink の新バージョンを入手したら、ドキュメントを確認して、見落としのないようにしてください。

このマニュアルの「Bink の概要」では、Bink の概要について説明します。当社のビデオ API を初めてご使用の場合は、この章からご覧ください。

マニュアルの次の 10 数セクションでは、サポートされる多くのプラットフォームのそれぞれについて Bink を説明します。Win32、Win64、ソニー プレイステーション 3、ソニー PSP、ソニー プレイステーション 2、Xbox 360、Xbox、ニンテンドー ゲームキューブ、ニンテンドー Wii、MacOS、最後に Linux があります。

次に、本ドキュメントのリファレンス セクションについて概説します。リファレンスでは、Bink API と Windows および MacOS ブリッティング API (BinkBuffer API) のすべての機能について説明します。

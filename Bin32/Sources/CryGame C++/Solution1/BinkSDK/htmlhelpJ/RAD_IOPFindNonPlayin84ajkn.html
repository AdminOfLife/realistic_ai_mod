RAD_IOPFindNonPlayingVoiceMask

関数 RAD_IOPFindNonPlayingVoiceMask は、1 つの非再生中音声に対応するビット マスクを返します。

S32 RAD_IOPFindNonPlayingVoiceMask(
  U32 mask
);

入力パラメータ

mask
再生中および非再生中音声のマスクを指定します (通常は、RAD_IOPQueueAsyncSPUGlobalState から返されます)。

戻り値

入力マスク内の非再生中音声の 1 つのビット マスクを返します。

説明

この関数は簡単なユーティリティ関数です。

この関数は、音声再生マスク (「1」のビットは音声中の再生に対応し、「0」のビットは非再生中の音声を示します) を受け取り、再生されていない最初の音声を示すマスクを返します。たとえば、音声 2 以外のすべての音声が再生されている場合、入力マスクは 0xfffffb になり、この関数は 4 (1 << 2) を返します。


グループ: ソニー PS2 用 RAD_IOP SPU ハードウェア音声 API
関連する関数: RAD_IOPQueueAsyncSPUGlobalState
関連する基本データ型: S32, U32

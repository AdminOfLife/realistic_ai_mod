MacOS プラットフォーム

説明

Bink が 2 番目にサポートを開始したオペレーティング システムが MacOS です。MacOS は、ある部分では先進的でありながら一部では時代に逆行するという特徴を持つため、開発も通常とは異なります。Bink は MacOS X の下で開発され、Intel と PowerPC の両方の Mac に対応する Mach-O バージョン ライブラリが追加されています。また、従来の MacOS および Carbon バージョンの Bink SDK も提供しています。

このプラットフォームでは、Bink の使用も統合もきわめて簡単です。必要なことはすべてこの章で説明します。

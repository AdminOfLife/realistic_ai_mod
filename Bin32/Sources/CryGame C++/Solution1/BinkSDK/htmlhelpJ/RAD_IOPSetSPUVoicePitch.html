Bink SDK 1.9c

RAD_IOPSetSPUVoicePitch

関数 RAD_IOPSetSPUVoicePitch は、1 つ以上の SPU 音声のピッチを設定します。

S32 RAD_IOPSetSPUVoicePitch(
  U32 core0_voice_mask,
  U32 core1_voice_mask,
  U32 pitch
);

入力パラメータ

core0_voice_mask
設定するコア 0 の音声のビット マスクを指定します。
core1_voice_mask
設定するコア 1 の音声のビット マスクを指定します。
pitch
使用するピッチを指定します。

戻り値

正常ならば 0 以外の値、失敗の場合は 0 を返します。

説明

この関数は、1 つ以上の SPU 音声のピッチを設定するように SPU に指示します。

ピッチはサウンドの周波数や再生レートと同じではありません。ピッチは「(周波数 * 4096 ) / 48000」と定義されます。したがって、再生レートを 22050 に設定するには、22050 * 4096 / 48000 = 1882 を使用します。

SPU 音声の再生を一時停止する最善の方法は、ピッチを 0 に設定することです。これにより、元のピッチを再設定して再開するまで、実質的に音声の再生位置が一時停止します。

関連する基本データ型: S32, U32

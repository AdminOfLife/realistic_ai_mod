Bink SDK 1.9c

Bink のオーディオ システムのインターフェイス関数

説明

Bink の統合をできるだけ簡単にするために、サポートしているすべてのプラットフォームでオーディオとのインターフェイス コードをあらかじめ作成して提供しています。異なる API やハードウェアをサポートできるように、複数のサウンド インターフェイスを持つプラットフォームもあります。

ユーザーのゲームで動作するサウンド インターフェイスがない場合、インターフェイスのコードを独自に開発することもできます (複雑な作業なので、詳細については当社にお問い合わせください)。ただし、これはきわめてまれなことです。

Bink のオーディオ システムのインターフェイス関数
BinkSoundUseMiles Miles Sound System をサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseLibAudio LibAutio を Sony PS3 のサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUsePSPSound Sony PSP で libwave またはシンプル オーディオをサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseDirectSound DirectSound をサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseWaveOut 非推奨の waveOut オーディオ インターフェイスをサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseRAD_IOP RAD_IOP API をサウンド出力システムとして使用するように Bink に指示します (PS2 の IOP チップを除く)。
BinkSoundUseSoundManager MacOS Sound Manager をサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseNDSSound ニンテンドー DS サウンド ハードウェアをサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseAX AX をニンテンドー ゲームキューブまたは Wii のサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseMusyX MusyX をニンテンドー ゲームキューブのサウンド出力システムとして使用するように Bink に指示します。
RADARAMCALLBACKS ニンテンドー ゲームキューブまたは Wii で AX を使用するために代用するオーディオ RAM メモリ管理用の関数を指定します。
RADARAMALLOC ニンテンドー ゲームキューブまたは Wii で AX を使用する場合に、Bink の代わりにオーディオ メモリを割り当てる関数をポイントします。
RADARAMFREE すでに RADARAMALLOC 関数を呼び出すことで割り当てられているメモリを解放する関数をポイントします。
BinkSoundUseSDLMixer SDL_mixer を GNU/Linux でのサウンド出力システムとして使用するように Bink に指示します。
BinkSetSoundSystem カスタムのサウンド プロバイダをサウンド出力システムとして使用するように Bink に指示します。

Bink SDK 1.9c

BinkSetMemory

BinkSetMemory 関数は、メモリ管理の関数の代用を指定します。

void BinkSetMemory(
  RADMEMALLOC mem_alloc,
  RADMEMFREE mem_free
);

入力パラメータ

mem_alloc
メモリを割り当てるために呼び出す関数を指定します。
mem_free
mem_alloc が割り当てたメモリを解放するために呼び出す関数を指定します。

説明

この関数を使用すると、メモリの割り当てや解放のために Bink が呼び出す関数を代用できます。

これらの関数では、32 バイト境界で揃えられたメモリ ポインタを返す必要があります。

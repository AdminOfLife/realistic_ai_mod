Bink SDK 1.9c

ニンテンドー ゲームキューブ用の Bink サンプル

説明

Bink SDK には、ニンテンドー ゲームキューブ用のプログラムのサンプルがいくつか用意されています。SN システムや CodeWarrior でこれらのサンプルを作成するには、新規プロジェクトを作成して適切な C のソース ファイル (ExamNGCbb.C または ExamNGCtex) をインクルードし、次にニンテンドー ゲームキューブ版 Bink のスタティック ライブラリをプロジェクト (BinkNGC.a) に追加するだけです。

練習用のサンプル ファイルは次のとおりです。

Bink を使用して、ビデオを直接ゲームキューブの YUV バック バッファに表示するサンプルについては、ファイル ExamNGCbb.C を参照してください。このサンプルは、ビデオのレンダリングに GPU を使用しないため、きわめてシンプルです。バック バッファに直接ブラストするだけです。また、YUV フォーマットから変換する必要もないため、処理速度も格段に向上します。

ニンテンドー GameCube の GPU ハードウェアを使用して、直接 Bink の内部 YUV バッファからレンダリングするサンプルについては、ExamNGCtex.C ファイルを参照してください。このサンプルでは 3D ラッパーを使用するので、必ずファイル NGCtextures.C もプロジェクトに追加してください。このサンプルは、通常の Bink ビデオまたはアルファ Bink ビデオをレンダリングします。

これらのサンプルでは、ゲームキューブを設定するために必ず 1 つの簡単なファイルを使用します。サンプルのプロジェクトには、NGCDemo.c も加えてください。

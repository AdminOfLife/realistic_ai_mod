Win32 でのメモリ管理

説明

デフォルトでは、Bink は Windows のメモリ機能を直接呼び出して、メモリ管理を処理します。ただし、Bink の割り当て関数や解放関数を BinkSetMemory 関数に置き換えることもできます。

BinkGetSummary 関数を使用すると、メモリの使用量や最大使用量を確認できます。

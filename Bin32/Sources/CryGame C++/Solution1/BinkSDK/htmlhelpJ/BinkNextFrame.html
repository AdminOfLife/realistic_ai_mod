Bink SDK 1.9c

BinkNextFrame

BinkNextFrame 関数は、Bink ファイルのビデオの次のフレームに移動します。

void BinkNextFrame(
  HBINK bink
);

入力パラメータ

bink
Bink ハンドルを指定します。

説明

この関数により、実際に次のフレームに移動します。通常はフレームを BinkDoFrame で解凍してからこの関数を呼び出し、画面にブリットします。

Bink ファイルの最後のフレームでこの関数を呼び出すと、Bink は最初のフレームに戻ってループを繰り返します。つまり、アニメーションを 1 回だけ再生するには (ループさせずに)、一番最後のフレームに特別なロジックを追加して、この関数を呼び出さないようにする必要があります (そうしないと、Bink がファイルの先頭に戻るための遅延がビデオの末尾で発生します)。たとえば、Bink ファイルを 1 回だけ再生する単純な再生ループは、次のとおりです。

while ( 1 )
{
  BinkDoFrame( bink );

  show_video_frame();

  while ( BinkWait( bink ) ) ;

  if ( bink->FrameNum == bink->Frames )  // ファイルの最後で BinkNextFrame ループから抜ける
    break;

  BinkNextFrame( bink );
}

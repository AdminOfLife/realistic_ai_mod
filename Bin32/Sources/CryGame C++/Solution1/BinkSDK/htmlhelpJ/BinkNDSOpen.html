Bink SDK 1.9c

BinkNDSOpen

BinkNDSOpen 関数は、指定された Bink ファイルを開いて解凍します。

HBINK BinkNDSOpen(
  FSFileID_PTR file_spec,
  BINK_OPEN_FLAGS open_flags
);

入力パラメータ

file_spec
開く対象である NDS の FSFileID へのポインタ。
open_flags
ファイルを開くときのフラグ。

戻り値

開かれた Bink ファイルのハンドル (エラーの場合は 0)

説明

この関数は BinkOpen と完全に同様に動作し、Nintendo DS の FSFileID を開くために使用します。

返される HBINK ハンドルは、開いているファイルの情報を含む BINK 構造体へのポインタでもあります。

BinkNDSOpen の呼び出しが失敗すると、0 が返されます。BinkGetError を呼び出すと、失敗についての説明を表示できます。

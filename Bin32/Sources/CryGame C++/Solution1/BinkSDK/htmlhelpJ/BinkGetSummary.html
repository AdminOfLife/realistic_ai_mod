Bink SDK 1.9c

BinkGetSummary

BinkGetSummary 関数は、指定した Bink ハンドルの再生情報を返します。

void BinkGetSummary(
  HBINK bink,
  BINKSUMMARY *summary
);

入力パラメータ

bink
Bink ハンドルを指定します。

出力パラメータ

summary
要約した情報を返す構造体を指定します。

説明

この関数は、再生したばかりの Bink ファイルに関するあらゆる有益な情報を返します。

通常、この関数は Bink ファイルを閉じる直前にを呼び出します。

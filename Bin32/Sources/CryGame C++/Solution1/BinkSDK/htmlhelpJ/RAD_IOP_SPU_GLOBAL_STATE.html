Bink SDK 1.9c

RAD_IOP_SPU_GLOBAL_STATE

構造体 RAD_IOP_SPU_GLOBAL_STATE は、RAD_IOPQueueAsyncSPUGlobalState 関数で使用される構造体です。

struct RAD_IOP_SPU_GLOBAL_STATE
{
  RAD_IOP_SPU_GLOBAL_STATE_CALLBACK callback;
  RAD_IOP_STATUS_VALUES status;
  S32 current_playing_mask[ 2 ];
  U32 current_playing_count[ 2 ];
  U32 current_dry_left_mask[ 2 ];
  U32 current_wet_right_mask[ 2 ];
  U32 current_wet_left_mask[ 2 ];
  U32 current_wet_right_mask[ 2 ];
  U32 current_end_hit_mask[ 2 ];
};

入力メンバ

callback
終了時に実行するコールバックを指定します。コールバックが不要な場合は、このフィールドに 0 を設定します。

出力メンバ

status
非同期ステータスの現在のステータスを示します。
current_playing_mask[ 2 ]
コア 0 の現在の再生中/非再生中マスクを最初の U32 に返し、コア 1 の同じ情報を 2 番目の U32 に返します。
current_playing_count[ 2 ]
コア 0 の現在の音声再生の数を最初の U32 に返し、コア 1 の同じ情報を 2 番目の U32 に返します。
current_dry_left_mask[ 2 ]
コア 0 のオン/オフ ドライ左チャンネル マスクを最初の U32 に返し、コア 1 の同じ情報を 2 番目の U32 に返します。
current_wet_right_mask[ 2 ]
コア 0 のオン/オフ ドライ右チャンネル マスクを最初の U32 に返し、コア 1 の同じ情報を 2 番目の U32 に返します。
current_wet_left_mask[ 2 ]
コア 0 のオン/オフ ウェット左チャンネル マスクを最初の U32 に返し、コア 1 の同じ情報を 2 番目の U32 に返します。
current_wet_right_mask[ 2 ]
コア 0 のオン/オフ ウェット右チャンネル マスクを最初の U32 に返し、コア 1 の同じ情報を 2 番目の U32 に返します。
current_end_hit_mask[ 2 ]
低レベル ENDX レジスタ マスクを返します。

説明

この構造体は、RAD_IOPQueueAsyncSPUGlobalState 関数の入力と出力の両方で使用されます。入力パラメータを提供すると、RAD_IOP が出力パラメータを返します。

current_playing_mask フィールドはビット マスクで、current_playing_count はその数です。したがって、音声 5 と 7 がコア 0 で再生されている場合、current_playing_mask[ 0 ] は「( 1 << 5 ) | ( 1 << 7 )」になり、current_playing_count[ 0 ] は 2 になります。

コールバックを設定していない場合は、ステータス フィールドを継続的にモニタして、このフィールドが RAD_IOP_STATUS_COMPLETE 以上になるのを待つことができます。次に、RAD_IOP_STATUS_DONE ステータスをクエリして、クエリが正常に行われたかどうかを判別します。

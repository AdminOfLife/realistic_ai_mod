Bink SDK 1.9c

BinkSetIOSize

BinkSetIOSize 関数は、Bink が読み込む I/O メモリのバッファ サイズを指定します。

void BinkSetIOSize(
  U32 io_size
);

入力パラメータ

io_size
Bink が先読み I/O バッファとして使用するサイズをバイト数で指定します。

説明

この関数を使用すると、Bink が Bink ファイルの先読みに使用する I/O バッファのサイズを設定できます。デフォルトでは、I/O バッファはムービーの中で最もサイズが大きい 1 秒あたりデータ レートです。

この関数を使用するには、最初にこの関数を呼び出して、次に BINKIOSIZE フラグを BinkOpen の呼び出しに追加します。


グループ: Bink API
関連項目: 再生時のゲーム データのロード
関連する関数: BinkOpen
関連する基本データ型: BINK_OPEN_FLAGS, U32, void

Bink SDK 1.9c

BinkOpen

BinkOpen 関数は、指定された Bink ファイルを開いて解凍します。

HBINK BinkOpen(
  CHAR_PTR file_name,
  BINK_OPEN_FLAGS open_flags
);

入力パラメータ

file_name
開くファイルの名前。
open_flags
ファイルを開くときのフラグ。

戻り値

開かれた Bink ファイルのハンドル (エラーの場合は 0)

説明

この関数は、Bink API のすべての関数が使用する HBINK ハンドルを返します。

返される HBINK ハンドルは、開いているファイルの情報を含む BINK 構造体へのポインタでもあります。

BinkOpen の呼び出しが失敗すると、0 が返されます。BinkGetError を呼び出すと、失敗についての説明を表示できます。

Bink SDK 1.9c

BinkDX8SurfaceType

関数 BinkDX8SurfaceType は、指定された DirectX 8 サーフェスのフォーマットを表す値を返します。

BINK_COPY_FLAGS BinkDX8SurfaceType(
  LPDIRECT3DSURFACE8 lpDirect3DSurface
);

入力パラメータ

lpDirect3DSurface
クエリするサーフェスを指定します。

戻り値

指定されたサーフェスのフォーマットを表す値を返します。

説明

この関数は、提供された DirectX 8 サーフェスを表す値を返します。この関数を使用してサーフェス タイプを取得し、BinkCopyToBuffer 関数に渡すことができます。

サーフェスのタイプが未定義またはサポートされていない場合、この関数は -1 を返します。

他のバージョンの DirectX の場合は、BinkDDSurfaceType または BinkDX9SurfaceType 関数を使用します。

MacOS では、BinkGDSurfaceType 関数を使用します。


グループ: BinkBuffer API
関連する関数: BinkCopyToBuffer, BinkDDSurfaceType, BinkDX9SurfaceType, BinkGDSurfaceType
関連する基本データ型: BINK_COPY_FLAGS, LPDIRECT3DSURFACE8

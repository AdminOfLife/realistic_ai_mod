Bink SDK 1.9c

BinkSoundUseLibAudio

BinkSoundUseLibAudio 関数は、LibAutio を Sony PS3 のサウンド出力システムとして使用するように Bink に指示します。

S32 BinkSoundUseLibAudio(
  U32 number_of_speakers
);

入力パラメータ

number_of_speakers
LibAudio を初期化して使用するスピーカーの数を指定します。

戻り値

Bink の出力レイヤが LibAudio に切り替わったかどうかを返します (正常に終了した場合は 0 以外、失敗した場合は 0)。

説明

この関数 (実際はマクロ) は、オーディオ出力としてソニー LibAudio システム API を使用するように Bink に指示します。

PS3 の Bink は、7.1 チャンネルまでのオーディオ出力 (5.1 チャンネル、ステレオなど、7.1 チャンネルまでの任意のフォーマット) をサポートします。この関数は、LibAudio の初期化に使用するスピーカーの数を Bink に指示します。通常、この値は、LibAudio の関数 cellAudioOutGetDeviceInfo を使用して取得されます。また、cellAudioOutConfigure を呼び出して、ダンプするスピーカーの数を PS3 ハードウェアにも指示してください。詳細については、ExamPS3.cpp ファイルを参照してください。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前に呼び出さなければ、この関数は失敗します。

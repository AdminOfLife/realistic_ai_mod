Bink SDK 1.9c

BinkGetError

BinkGetError 関数は、最新のグローバル Bink エラーを説明する文字列を返します。

CHAR_PTR BinkGetError();

戻り値

最新のグローバル Bink エラーを返します。

説明

この関数は、最新のグローバル Bink エラーを説明する文字を返します。これは、BinkOpen が失敗した場合に特に便利です。


グループ: Bink API
関連する関数: BinkMacOpen, BinkNDSOpen, BinkOpen
関連する基本データ型: CHAR_PTR

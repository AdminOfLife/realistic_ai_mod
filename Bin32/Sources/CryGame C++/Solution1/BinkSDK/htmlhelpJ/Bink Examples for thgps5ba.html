Bink SDK 1.9c

Xbox 360 用の Bink サンプル

説明

Bink SDK には、Xbox 360 用のサンプル プログラムがいくつか用意されています。Visual C でこれらのサンプルを作成するには、新規プロジェクトを作成して適切な C のソース ファイル (ExamXenon.CPP など) をインクルードし、次に Xbox 360 用の Bink のスタティック ライブラリをプロジェクト (BinkXenon.LIB または BinkXenon.LTCG.LIB) に追加するだけです。

練習用のサンプル ファイルは次のとおりです。

Direct3D ピクセル シェーダを使用して Bink ビデオを表示するサンプルについては、ファイル ExamXenon.CPP を参照してください。このサンプルでは Direct3D ラッパーを使用するので、必ずファイル BinkTextures.CPP もプロジェクトに追加してください。

Bink のアルファ平面で Direct3D ピクセル シェーダを使用するサンプルについては、ExamXenA.CPP ファイルを参照してください。このサンプルでは Direct3D ラッパーを使用するので、必ずファイル BinkTextures.CPP もプロジェクトに追加してください。

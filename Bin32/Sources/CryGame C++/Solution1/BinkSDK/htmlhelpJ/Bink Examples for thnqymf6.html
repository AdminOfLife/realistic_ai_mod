Bink SDK 1.9c

ニンテンドー DS 用の Bink サンプル

説明

Bink SDK には、ニンテンドー DS 用の簡単なサンプル プログラムが用意されています。CodeWarrior を使用してこのサンプルを作成するには、新規プロジェクトを作成し、適切な C のソース ファイル (ExamNDS.C) をインクルードし、次に DS 版 Bink のスタティック ライブラリをプロジェクト (BinkNDS.a) に追加するだけです。

ExamNDS.C は、Bink ビデオを可能なかぎり簡単に再生します。このファイルの先頭で、8 ビットまたは 16 ビット ピクセル出力を使用するように設定できます。16 ビット ビデオのほうが高品質なので、この定義はそのままにしておくことをお勧めします。

Bink SDK 1.9c

BinkRestoreCursor

関数 BinkRestoreCursor は、BinkCheckCursor の呼び出しの後で、マウス カーソルを元の状態に復元します。

void BinkRestoreCursor(
  S32 check_count
);

入力パラメータ

check_count
BinkCheckCursor 呼び出しから返されるカウント値です。

説明

この関数は、BinkCheckCursor の呼び出しの後で、マウス カーソルを元の状態に復元します。

この関数は、プライマリ サーフェスのロックを解除してから呼び出す必要があります。そうしないと、カーソルは復元されません。

この関数の使用例については、BinkIsSoftwareCursor 関数を参照してください。


グループ: BinkBuffer API
関連項目: マウス カーソル
関連する関数: BinkCheckCursor, BinkIsSoftwareCursor
関連する基本データ型: S32, void

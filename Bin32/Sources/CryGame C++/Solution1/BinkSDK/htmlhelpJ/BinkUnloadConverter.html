Bink SDK 1.9c

BinkUnloadConverter

BinkUnloadConverter 関数は、Bink YUV から RGB への少なくとも 1 つの変換をアンロードします。

void BinkUnloadConverter(
  BINK_CONVERTER_FLAGS flags
);

入力パラメータ

flags
アンロードする変換を指定します。

説明

この関数は、Bink の 1 つまたは複数の YUV 変換をメモリからアンロードします。ゲームで実際に使用する YUV 変換だけをロードすることで、メモリを効果的に使用することができます。たとえば、サーフェスの種類に BINKSURFACE32 と BINKSURFACE32A を使用する場合は、次のようなコードを使用して最小限の変換だけをロードします。

// 最初に、すべての変換をアンロード
BinkUnloadConverter( BINKCONVERTERSALL );

// 次に、使用する変換だけをロード
BinkLoadConverter( BINKSURFACEYUY2 );
BinkLoadConverter( BINKSURFACE32A );


グループ: Bink API
関連項目: Xbox のメモリ管理
関連する関数: BinkLoadConverter, BinkUnload
関連する基本データ型: BINK_CONVERTER_FLAGS, BINK_COPY_FLAGS, void

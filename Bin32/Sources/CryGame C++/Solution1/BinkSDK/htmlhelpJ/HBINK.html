Bink SDK 1.9c

HBINK

基本データ型 HBINK は、開いている Bink ファイルのハンドルです。

説明

HBINK ハンドルは、BinkOpen 関数から返されます。

また、開いている Bink ファイルに関する主要なすべての情報を含む BINK 構造体をポイントします。

BinkOpen の呼び出しが失敗した場合、このハンドルは 0 になります。


グループ: 基本データ型
関連する関数: BinkClose, BinkControlBackgroundIO, BinkCopyToBuffer, BinkCopyToBufferRect, BinkDoFrame, BinkGetFrameBuffersInfo, BinkGetKeyFrame, BinkGetRealtime, BinkGetRects, BinkGetSummary, BinkGetTrackID, BinkGoto, BinkMacOpen, BinkNDSOpen, BinkNextFrame, BinkOpen, BinkOpenTrack, BinkPause, BinkRegisterFrameBuffers, BinkService, BinkSetMixBinVolumes, BinkSetMixBins, BinkSetPan, BinkSetSoundOnOff, BinkSetVideoOnOff, BinkSetVolume, BinkShouldSkip, BinkWait
関連する構造体: BINK

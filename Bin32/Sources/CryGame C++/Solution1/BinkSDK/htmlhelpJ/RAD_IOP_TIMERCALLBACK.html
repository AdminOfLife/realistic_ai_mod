Bink SDK 1.9c

RAD_IOP_TIMERCALLBACK

関数 RAD_IOP_TIMERCALLBACK は、定期的に呼び出される関数をポイントします。

void RAD_IOP_TIMERCALLBACK(
  RAD_IOP_TIMER_DATA const *timer_data
);

入力パラメータ

timer_data
RAD_IOPRegisterTimerCallback を呼び出したときの構造体へのポインタを指定します。

説明

この関数は、RAD_IOPStartTimer 関数が呼び出されると、1 回呼び出されます。コールバックは、インタラプト時に実行されるため、この関数内に作成するコードはインタラプト セーフにする必要があります。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPRegisterTimerCallback, RAD_IOPStartTimer
関連する基本データ型: void
関連する構造体: RAD_IOP_TIMER_DATA

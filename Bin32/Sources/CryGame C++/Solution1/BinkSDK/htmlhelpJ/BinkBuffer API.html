Bink SDK 1.9c

BinkBuffer API

説明

BinkBuffer API は、アプリケーションに YUV ブリッティングを簡単に追加できるようにするために作成されました。Win32 では、BinkBuffer API は基本的な 5 種類のブリッティング スタイル (プライマリ、DIBSection、オーバーレイ、YUV オフスクリーン、および RGB オフスクリーン) をサポートし、これらのスタイル間を簡単に切り替えることができます。MacOS では、ブリッティング スタイルとして画面に直接出力する方法 (direct-to-screen) と GWorld をサポートします。BinkBuffer API は Win32 と MacOS でほとんど同じなので、移植は容易です。BinkBuffer API は、Windows と MacOS でのみサポートされます。

BinkBuffer API は、ほぼ即時にアプリケーションを起動して実行します。ただし、この動作はオプションです。すでに DirectDraw、Direct3D、または独自のブリッティング ソリューションを起動して実行している場合、BinkBuffer API を完全に省略して、Bink API の著作権だけをサーフェス ポインタに加えることができます。

また、3D システムがアクティブな場合は BinkBuffer API を使用しないように注意してください。BinkBuffer API は比較的新しい 3D カードでは作動しますが、古いカード (または古いドライバ) では 2D データを 3D ウィンドウにブリットできないため、画面が表示されません。

BinkBuffers は、必ずしもシステム メモリを追加で使用するとはかぎりません。オーバーレイとほとんどのオフスクリーン サーフェスはビデオ メモリに置かれ、プライマリ サーフェス BinkBuffer によるメモリの追加使用はまったく必要ありません。

BinkBuffer は Bink API とほぼ同様に使用されます。HBINKBUFFER ハンドル (BINKBUFFER 構造体へのポインタ) を開いて受け取り、このハンドルを BinkBuffer API 関数で使用し、完了したら BinkBuffer を閉じます。簡単なサンプルとして、ExamWin.C または ExampleMac.c ファイルを参照してください。

BinkBuffer API
BinkBufferOpen BinkBuffer (YUV データの画面へのブリッティングを容易にします) を開きます。
BINKBUFFER BinkBufferOpen から返される構造体です。
BINKBUFFER_OPEN_FLAGS 開こうとする BinkBuffer のタイプと機能を表します。
BinkBufferClose 指定された BinkBuffer ハンドルを閉じます。
BinkBufferSetDirectDraw DirectDraw オブジェクトとプライマリ サーフェス ポインタを設定します。
BinkBufferLock BinkBuffer によるピクセル バッファの読み取りまたは書き込み時にバッファをロックします。
BinkBufferUnlock BinkBufferLock によってロックされていた BinkBuffer のロックを解除します。
BinkBufferBlit ピクセル データを画面にコピーします (オフスクリーンの各種の BinkBuffer 用)。
BinkBufferCheckWinPos 指定されたウィンドウの位置座標を現在の BinkBuffer タイプに適合するように調整します。
BinkBufferSetOffset デスクトップ上のウィンドウの位置またはウィンドウ内の再生オフセットが変更されたことを BinkBuffer に通知します。
BinkBufferSetScale ブリッティング中に使用する新しい縮小または拡大倍率を設定します。
BinkBufferSetResolution BinkBuffer が開かれるときにビデオの解像度を切り替えるように Bink に指示します。
BinkBufferGetDescription BinkBuffer が使用しているブリッティング スタイルを表す文字列を返します。
BinkBufferGetError 最新のグローバル BinkBuffer エラーを説明する文字列を返します。
BinkDDSurfaceType 指定された DirectDraw サーフェスのフォーマットを表す値を返します。
BinkDX8SurfaceType 指定された DirectX 8 サーフェスのフォーマットを表す値を返します。
BinkDX9SurfaceType 指定された DirectX 9 サーフェスのフォーマットを表す値を返します。
BinkGDSurfaceType 指定された MacOS のグラフィック デバイスのフォーマットを表す値を返します。
BinkIsSoftwareCursor 指定された DirectDraw サーフェス上で、指定されたマウス カーソルがソフトウェア ベースのカーソルであるかどうかを判別します。
BinkCheckCursor 現在のマウスの位置が指定された長方形のウィンドウの内側であるかどうかをチェックし、内側の場合はマウスを表示しないようにします。
BinkRestoreCursor BinkCheckCursor の呼び出しの後で、マウス カーソルを元の状態に復元します。

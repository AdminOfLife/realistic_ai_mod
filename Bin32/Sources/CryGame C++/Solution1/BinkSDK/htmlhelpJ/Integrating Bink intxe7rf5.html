Bink SDK 1.9c

MacOS ビルド環境への Bink の組み込み

説明

Bink Video を使用するアプリケーションをビルドするには、ビルド プロセスに Bink SDK を含めます。それには、次の 3 つの簡単な手順に従います。

1) ソース ファイルで Bink.h をインクルードします。

2) BinkClassicLibrary、BinkCarbonLibrary、libBinkMachO.dylib、または libBinkMachOx86.dylib ライブラリ ファイルにリンクします (Xcode または CodeWarrior で、ライブラリをプロジェクトにドラッグするだけです)。

3) アプリケーションの実行可能ファイルがあるフォルダに、BinkClassicLibrary、BinkCarbonLibrary、libBinkMachO.dylib、libBinkMachOx86.dylib のいずれかのライブラリ ファイルを置きます。MachO アプリケーションでは、アプリケーションの実行可能ファイルが Contents/MacOS フォルダに置かれていることに注意してください。MachO アプリケーション パッケージは、単なる定義済みフォルダ セットです。したがって、Finder で Contents サブフォルダを開くには、Control キーを押しながらアプリケーションのアイコンをクリックし、[Show Package Contents] を選択します。次に、libBinkMachO.dylib または libBinkMach0x86.dylib ファイルを Contents/MacOS フォルダに入れます。

以上です。これで Bink を使用できるようになりました。

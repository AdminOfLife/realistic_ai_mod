Bink SDK 1.9c

RAD_IOPCancelGroupOfAsyncs

関数 RAD_IOPCancelGroupOfAsyncs は、指定されたグループ ID による今までのすべての非同期ファイル オープンと読み取りを取り消します。

S32 RAD_IOPCancelGroupOfAsyncs(
  U32 group
);

入力パラメータ

group
オープンと読み取り要求を取り消すグループ ID のハンドルを指定します。

戻り値

取り消された非同期処理の数を返します (0 の場合、通常はすでに取り消しが完了していることを示します)。

説明

この関数を使用すると、特定のグループ ID に基づいて、すでにキューに置かれているすべてのオープンと読み取り要求を取り消すことができます。ただし、コールバックはオープン要求が取り消されたときも実行されるため、必ずステータス フィールドを確認してください。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPCancelAllAsyncs, RAD_IOPCancelAsyncOpen, RAD_IOPCancelAsyncRead, RAD_IOPCancelFileAsyncs
関連する基本データ型: S32, U32

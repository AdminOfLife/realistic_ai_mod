Bink SDK 1.9c

BinkGetRealtime

BinkGetRealtime 関数は、指定した Bink ハンドルの再生情報をリアルタイムに返します。

void BinkGetRealtime(
  HBINK bink,
  BINKREALTIME *real_time,
  U32 frame_window_length
);

入力パラメータ

bink
Bink ハンドルを指定します。
frame_window_length
リアルタイム情報を返すフレーム ウィンドウの長さ (フレーム数) を指定します。

出力パラメータ

real_time
情報をリアルタイムに返す構造体を指定します。

説明

この関数は、現在再生している Bink ファイルのリアルタイム情報を返します。ムービーを見ながら、Bink の正確な再生状況を表示できます。

パラメータ frame_window_length を設定することにより、1 つ以上のフレームに対応する時間のリアルタイム情報を確認できます。

単純な使用法としては、パラメータ frame_window_length には 0 だけを使用します (最新の 1 秒分のフレームについてレポートするように Bink に指示します)。

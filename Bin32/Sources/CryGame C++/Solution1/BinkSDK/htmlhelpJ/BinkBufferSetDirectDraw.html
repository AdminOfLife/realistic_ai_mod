Bink SDK 1.9c

BinkBufferSetDirectDraw

関数 BinkBufferSetDirectDraw は、DirectDraw オブジェクトとプライマリ サーフェス ポインタを設定します。

void BinkBufferSetDirectDraw(
  LPDIRECTDRAW lpDirectDraw,
  LPDIRECTDRAWSURFACE lpddsPrimary
);

入力パラメータ

lpDirectDraw
使用する DirectDraw オブジェクト ポインタを指定します。
lpddsPrimary
使用する DirectDraw プライマリ サーフェス オブジェクト ポインタを指定します。

説明

Bink は、デフォルトで Bink 独自の DirectDraw およびプライマリ サーフェス オブジェクトを作成します。この関数は、アプリケーションですでに DirectDraw が起動され、実行されている場合に使用します。

したがって、ユーザー独自の DirectDraw オブジェクトがすでに作成されている場合、最初に BinkBufferOpen を呼び出す前に、この関数を呼び出す必要があります。


グループ: BinkBuffer API
関連する関数: BinkBufferOpen
関連する基本データ型: LPDIRECTDRAW, LPDIRECTDRAWSURFACE, void

基本データ型
S8 符号付き 8 ビット整数を表します。
U8 符号なし 8 ビット整数を表します。
S16 符号付き 16 ビット整数を表します。
U16 符号なし 16 ビット整数を表します。
S64 符号付き 64 ビット整数を表します。
U64 符号なし 64 ビット整数を表します。
S32 符号付き 32 ビット整数を表します。
U32 符号なし 32 ビット整数を表します。
s32_ptr 符号付き 32 ビット整数へのポインタを表します。
u32_ptr 符号なし 32 ビット整数へのポインタを表します。
VOID_PTR 標準の C/C++ の untyped メモリ ポインタを表します。
F32 標準の 32 ビット浮動小数点型の数値を表します。
F64 標準の 64 ビット倍精度浮動小数点型の数値を表します。
void 標準の C/C++ の void 戻り型を表します。
NULL 標準の C/C++ の NULL ポインタを表します。
OS_WND Win32 上の標準の HWND ウィンドウ ハンドル、または MacOS での標準の WindowPtr を表します。
HCURSOR Win32 の HCURSOR カーソル ハンドルを表します。
LPDIRECTSOUND DirectSound オブジェクトへのポインタを表します。
LPDIRECTDRAW DirectDraw オブジェクトへのポインタを表します。
LPDIRECTDRAWSURFACE DirectDraw サーフェス オブジェクトへのポインタを表します。
LPDIRECT3DSURFACE8 DirectX 8 Direct3D サーフェス オブジェクトへのポインタを表します。
LPDIRECT3DSURFACE9 DirectX 9 Direct3D サーフェス オブジェクトへのポインタを表します。
FSSpec_PTR MacOS の filespec へのポインタを表します。
FSFileID_PTR ニンテンドー DS FSFileID へのポインタを表します。
GDHandle 標準の MacOS のグラフィック デバイスのポインタを表します。
VOID_PTR 標準の C/C++ の untyped メモリ ポインタを表します。
CHAR_PTR 0 で終わる通常の C 形式の文字列を表します。
HBINK 開いている Bink ファイルのハンドルです。
HBINKTRACK 開いている Bink のオーディオ トラックのハンドルです。
HBINKBUFFER 開いている BinkBuffer のハンドルです。
BINKRECT 長方形の領域を表す構造体です。
BINKRECT_ARRAY 長方形の領域を表す構造体の配列です。
Bink API
BinkOpen 指定された Bink ファイルを開いて解凍します。
BinkMacOpen 指定された Bink ファイルを開いて解凍します。
BinkNDSOpen 指定された Bink ファイルを開いて解凍します。
BINK BinkOpen から返される構造体です。
BINK_OPEN_FLAGS Bink ファイルを開く特殊な方法を表します。
BinkClose 指定された Bink ハンドルを閉じます。
BinkDoFrame ビデオの次のフレームを解凍します。
BinkCopyToBuffer Bink 内のメモリ バッファから指定されたメモリ アドレスにビデオのフレームをコピーします。
BinkCopyToBufferRect Bink 内のメモリ バッファから指定されたメモリ アドレスにビデオのフレームの一部をコピーします。
BINK_COPY_FLAGS Bink 内の YUV データの特殊な変換方法や変換先のバッファのタイプを表します。
BinkNextFrame Bink ファイルのビデオの次のフレームに移動します。
BinkWait 次の Bink フレームに移動する時間になったことを (ファイルのフレーム レートに基づいて) 通知します。
BinkPause Bink ファイルの再生を一時停止および再開します。
BinkGetRects 最後に解凍したビデオ フレームの変更された長方形領域を返します。
BINK_GETRECTS_FLAGS BinkGetRects 関数で使用する長方形の一覧の型を指定します。
BinkGoto Bink ファイル内の新しいフレームに移動します。
BINK_GOTO_FLAGS 実行する移動の種類を指定します。
BinkGetFrameBuffersInfo Bink が Y 平面、cR 平面、および cB 平面を保存する内部フレーム バッファに関する詳細を返します。
BINKFRAMEBUFFERS BinkGetFrameBuffersInfo の出力および BinkRegisterFrameBuffers への入力として使用される構造体です。
BINKFRAMEPLANESET 平面のセット (Y、cR、cB、およびアルファに対して 1 つ) を記述する構造体です。
BINKPLANE 1 つの 2D 平面を記述する構造体です。
BinkRegisterFrameBuffers Y、cR、および cB 平面を解凍するために Bink が使用するフレーム バッファ ポインタを設定します。
BinkGetKeyFrame 指定されたフレームの付近にあるキー フレームを検索します。
BINK_GETKEY_FLAGS 実行する検索の種類を指定します。
BinkShouldSkip フレームまたはブリットをスキップする必要があるほどビデオがオーディオより遅れているかどうかを報告します。
BinkService Bink がサウンド システムにサービスを提供するための明示的なアイドル時間を提供します。
BinkSetVolume Bink のサウンド トラックの音量を設定します。
BinkSetPan Bink のサウンド トラックのスピーカーのパン (右対左のバランス) を設定します。
BinkSetMixBins Xbox または Xbox 360 のサウンド トラックのミックス ビンの出力先を設定します。
BinkSetMixBinVolumes Xbox、Xbox 360、ソニー PS3、または Miles Sound System の個々のミックス ビンの音量を設定します。
BinkSetVideoOnOff BinkDoFrame を呼び出したときにビデオ フレームを解凍しないように Bink に指示します。
BinkSetSoundOnOff オーディオの解凍を禁止し、オーディオのリソースを解放するように Bink に指示します。
BinkGetSummary 指定された Bink ハンドルに関する再生情報を返します。
BINKSUMMARY 再生情報の要約を BinkGetSummary から返すために使用する構造体です。
BinkGetRealtime 指定された Bink ハンドルに関するリアルタイムの再生情報を返します。
BINKREALTIME リアルタイムの再生情報を BinkGetRealtime から返すために使用する構造体です。
BinkControlBackgroundIO Bink のバックグラウンド I/O システムを一時停止または再開します。
BINK_BG_CONTROL_FLAGS 切り替え先の I/O 状態を指定します。
BinkGetPalette BINKSURFACEP8 サーフェス タイプのグローバル Bink パレットを返します。
BinkGetTrackID 指定したトラック インデックスのトラック ID を返します。
BinkOpenTrack 低レベルのオーディオ トラック デコーダのハンドルを開きます。
BINKTRACK BinkOpenTrack から返される構造体です。
BinkCloseTrack 低レベルのオーディオ トラック デコーダのハンドルを閉じます。
BinkGetTrackData 現在のビデオ フレームからすべてのサウンドを抽出します。
BinkSetFrameRate Bink ファイルに組み込まれたフレーム レートを上書きします。
BinkSetSoundTrack Bink ファイルのデフォルトのサウンド トラックを上書きします。
BinkSetSimulate 指定されたデバイスへの転送速度をシミュレートするように Bink に指示します。
BinkSetIOSize Bink が読み込む I/O メモリのバッファ サイズを指定します。
BinkSetIO 置き換えるファイル I/O 関数のセットを指定します。
BinkSetMemory 置き換えるメモリ管理関数のセットを指定します。
RADMEMALLOC Bink にメモリを割り当てる関数をポイントします。
RADMEMFREE すでに RADMEMALLOC 関数を呼び出すことで割り当てられているメモリを解放する関数をポイントします。
BinkGetError 最新のグローバル Bink エラーを説明する文字列を返します。
BinkLogoAddress 埋め込まれた Bink ロゴ ファイルへのポインタを返します。
BinkLoad Xbox 上にある Bink のセクションを (BinkUnload でアンロードされていた場合) リロードします。
BinkUnload Xbox で Bink のセクションをアンロードします。
BinkLoadConverter Bink YUV から RGB への少なくとも 1 つの変換を (BinkUnloadConverter でアンロードされていた場合) リロードします。
BinkUnloadConverter は、Bink YUV から RGB への少なくとも 1 つの変換をアンロードします。
BINK_CONVERTER_FLAGS Bink YUV 変換のクラスをロードするかどうかを指定します。
Bink のオーディオ システムのインターフェイス関数
BinkSoundUseMiles Miles Sound System をサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseLibAudio LibAutio を Sony PS3 のサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUsePSPSound Sony PSP で libwave またはシンプル オーディオをサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseDirectSound DirectSound をサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseWaveOut 非推奨の waveOut オーディオ インターフェイスをサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseRAD_IOP RAD_IOP API をサウンド出力システムとして使用するように Bink に指示します (PS2 の IOP チップを除く)。
BinkSoundUseSoundManager MacOS Sound Manager をサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseNDSSound ニンテンドー DS サウンド ハードウェアをサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseAX AX をニンテンドー ゲームキューブまたは Wii のサウンド出力システムとして使用するように Bink に指示します。
BinkSoundUseMusyX MusyX をニンテンドー ゲームキューブのサウンド出力システムとして使用するように Bink に指示します。
RADARAMCALLBACKS ニンテンドー ゲームキューブまたは Wii で AX を使用するために代用するオーディオ RAM メモリ管理用の関数を指定します。
RADARAMALLOC ニンテンドー ゲームキューブまたは Wii で AX を使用する場合に、Bink の代わりにオーディオ メモリを割り当てる関数をポイントします。
RADARAMFREE すでに RADARAMALLOC 関数を呼び出すことで割り当てられているメモリを解放する関数をポイントします。
BinkSoundUseSDLMixer SDL_mixer を GNU/Linux でのサウンド出力システムとして使用するように Bink に指示します。
BinkSetSoundSystem カスタムのサウンド プロバイダをサウンド出力システムとして使用するように Bink に指示します。
BinkBuffer API
BinkBufferOpen BinkBuffer (YUV データの画面へのブリッティングを容易にします) を開きます。
BINKBUFFER BinkBufferOpen から返される構造体です。
BINKBUFFER_OPEN_FLAGS 開こうとする BinkBuffer のタイプと機能を表します。
BinkBufferClose 指定された BinkBuffer ハンドルを閉じます。
BinkBufferSetDirectDraw DirectDraw オブジェクトとプライマリ サーフェス ポインタを設定します。
BinkBufferLock BinkBuffer によるピクセル バッファの読み取りまたは書き込み時にバッファをロックします。
BinkBufferUnlock BinkBufferLock によってロックされていた BinkBuffer のロックを解除します。
BinkBufferBlit ピクセル データを画面にコピーします (オフスクリーンの各種の BinkBuffer 用)。
BinkBufferCheckWinPos 指定されたウィンドウの位置座標を現在の BinkBuffer タイプに適合するように調整します。
BinkBufferSetOffset デスクトップ上のウィンドウの位置またはウィンドウ内の再生オフセットが変更されたことを BinkBuffer に通知します。
BinkBufferSetScale ブリッティング中に使用する新しい縮小または拡大倍率を設定します。
BinkBufferSetResolution BinkBuffer が開かれるときにビデオの解像度を切り替えるように Bink に指示します。
BinkBufferGetDescription BinkBuffer が使用しているブリッティング スタイルを表す文字列を返します。
BinkBufferGetError 最新のグローバル BinkBuffer エラーを説明する文字列を返します。
BinkDDSurfaceType 指定された DirectDraw サーフェスのフォーマットを表す値を返します。
BinkDX8SurfaceType 指定された DirectX 8 サーフェスのフォーマットを表す値を返します。
BinkDX9SurfaceType 指定された DirectX 9 サーフェスのフォーマットを表す値を返します。
BinkGDSurfaceType 指定された MacOS のグラフィック デバイスのフォーマットを表す値を返します。
BinkIsSoftwareCursor 指定された DirectDraw サーフェス上で、指定されたマウス カーソルがソフトウェア ベースのカーソルであるかどうかを判別します。
BinkCheckCursor 現在のマウスの位置が指定された長方形のウィンドウの内側であるかどうかをチェックし、内側の場合はマウスを表示しないようにします。
BinkRestoreCursor BinkCheckCursor の呼び出しの後で、マウス カーソルを元の状態に復元します。
ソニー PS2 用 RAD_IOP API
RAD_IOPStartUp 使用する RAD_IOP API を開きます。
SOUND_CORE_FLAGS RAD_IOP が使用するサウンド フラグを指定します。
RAD_IOPMemoryAmount RAD_IOP が必要とする IOP メモリの大きさをバイト単位で返します。
RAD_IOPShutDown RAD_IOP ライブラリを閉じます。
RAD_IOPQueueAsyncOpenFile ファイルのハンドルを非同期で開きます。

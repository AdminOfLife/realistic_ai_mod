Bink SDK 1.9c

BINKTRACK

BINKTRACK 構造体は、BinkOpenTrack から返される構造体です。

struct BINKTRACK
{
  U32 Frequency;
  U32 Bits;
  U32 Channels;
  U32 MaxSize;
};

出力メンバ

Frequency
オーディオ トラックのサウンドの周波数を指定します。
Bits
オーディオ トラックのビット幅 (8 または 16) を指定します。
Channels
オーディオ トラックのチャンネル数 (1 = モノラル、2 = ステレオ) を指定します。
MaxSize
ムービーのいずれかのフレームに保存されているサウンド データの最大容量をバイト数で指定します (通常はフレーム 1 が最大)。BinkGetTrackData に渡すアドレスに、この大きさのバッファを割り当てる必要があります。

説明

BINKTRACK 構造体は、BinkOpenTrack が返す HBINKTRACK ハンドルによってポイントされます。ポインタを逆参照することにより、これらの情報フィールドにアクセスできます。

これらの値は、決して変更しないように注意してください。


グループ: Bink API
関連する関数: BinkGetTrackData, BinkOpenTrack
関連する基本データ型: HBINKTRACK, U32

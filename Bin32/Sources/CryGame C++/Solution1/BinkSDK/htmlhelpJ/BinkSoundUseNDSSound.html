Bink SDK 1.9c

BinkSoundUseNDSSound

BinkSoundUseNDSSound 関数は、ニンテンドー DS のサウンド ハードウェアをサウンド出力システムとして使用するように Bink に指示します。

S32 BinkSoundUseNDSSound(
  U32 channels_and_alarms
);

入力パラメータ

channels_and_alarms
使用できるサウンド チャンネルとサウンド アラームを Bink に指示するビットフィールド値。

戻り値

Bink の出力レイヤが DS サウンドに切り替わったかどうかを返します (正常に終了した場合は 0 以外、失敗した場合は 0)。

説明

この関数 (実際はマクロ) は、オーディオ出力としてニンテンドー DS サウンド ハードウェアを使用するように Bink に指示します。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前に呼び出さなければ、この関数は失敗します。

channels_and_alarms パラメータは、使用できるサウンド チャンネルとサウンド アラームを Bink に指示します。Bink は、ムービーの再生時にこれらのリソースだけを使用します。同時に再生される Bink ムービーごとに、少なくとも 1 つのサウンド チャンネル (ステレオの場合は 2 つ) と 1 つのサウンド アラームが必要です。ムービーの再生中にゲームで使用されないチャンネルとアラームを選択してください。ビットフィールド値を指定するには、2 つの単純なヘルパー マクロを使用します。たとえば、チャンネル 4 とアラーム 1 を使用する場合は、次のように指定します。


BinkSoundUseNDSSound( BINK_PACK_CHANNEL( 4 ) | BINK_PACK_ALARM( 1 ) );

ステレオ出力用にチャンネル 4 および 5 とアラーム 2 を使用する場合は、次のように指定します。


BinkSoundUseNDSSound( BINK_PACK_CHANNEL( 4 ) | BINK_PACK_CHANNEL( 5 ) | BINK_PACK_ALARM( 2 ) );


グループ: Bink のオーディオ システムのインターフェイス関数
関連項目: ニンテンドー DS のサウンド

Bink SDK 1.9c

BinkSetVideoOnOff

BinkSetVideoOnOff 関数は、BinkDoFrame を呼び出したときにビデオ フレームを解凍しないように Bink に指示します。

S32 BinkSetVideoOnOff(
  HBINK bink,
  S32 on_off
);

入力パラメータ

bink
Bink ハンドルを指定します。
on_off
ビデオを解凍するか (0 以外)、オーディオだけを解凍するか (0) を指定します。

戻り値

ビデオが現在オンか (0 以外) オフか (0) を返します。

説明

この関数を使用すると、再生している Bink ファイルのビデオの解凍を中止できます。これは、Bink ファイルのオーディオ トラックだけを再生していて、ビデオ トラックの解凍を考慮しない場合に便利です。

この関数はトグルなので、一度 0 で呼び出してビデオをオフにし、その後 0 以外の値で呼び出してビデオの再生を再開します。

この関数は、ビデオがオンの場合は 0 以外、ビデオがオフの場合は 0 を返します。

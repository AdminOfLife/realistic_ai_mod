Bink SDK 1.9c

ソニー プレイステーション 3 プラットフォーム

説明

Bink は、ソニーが新しく開発したプレイステーション 3 にも対応しています。ソニー PS3 は高速で、優れた GPU を備えており、美しいビデオを再生できます。ほとんどの開発者は、PS3 に HD 1280x720 ビデオを組み込むと思われます。

このプラットフォームに対して多くの最適化を行いました。PS3 では、PPC アセンブリ、Linux、および OpenGL の経験をうまく組み合わせる必要がありました。GPU を使いこなすには時間がかかりましたが、これができるようになると、フラグメント シェーダによってビデオの見栄えが素晴らしくなりました。

このプラットフォームでは、Bink の使用も統合もきわめて簡単です。必要なことはすべてこの章で説明します。

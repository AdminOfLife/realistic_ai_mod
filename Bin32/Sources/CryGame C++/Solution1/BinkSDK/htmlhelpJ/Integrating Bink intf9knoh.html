Bink SDK 1.9c

Win64 ビルド環境への Bink の組み込み

説明

Bink Video を使用するアプリケーションをビルドするには、ビルド プロセスに Bink SDK を含めます。それには、次の 4 つの簡単な手順に従います。

1) プロジェクトのインクルード ディレクトリとライブラリ ディレクトリに Bink SDK ディレクトリを追加します。

2) ソース ファイルで Bink.h をインクルードします。

3) インポート ライブラリ BinkW64.lib にリンクします。

4) アプリケーションをインストールしたディレクトリに、ファイル BinkW64.DLL を置きます。

以上です。これで Bink を使用できるようになりました。

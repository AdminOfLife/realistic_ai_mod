Bink SDK 1.9c

ソニー PSP 用の Bink サンプル

説明

Bink SDK には、ソニー PSP 用の完全なサンプル プログラムが用意されています。このサンプルを作成するには、新規プロジェクトを作成し、C のソース ファイル (ExamPSP.C、PageFlip.C、および BinkGuTextures.C) をインクルードし、次にソニー PSP 版 Bink のスタティック ライブラリをプロジェクト (BinkPSP.a) に追加するだけです。

このサンプルは、PSP の 3D ソフトウェアを使用して通常のビデオまたはアルファ ビデオを描画する方法を示します。

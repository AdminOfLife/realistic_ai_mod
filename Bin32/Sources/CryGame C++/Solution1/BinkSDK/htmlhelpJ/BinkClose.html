Bink SDK 1.9c

BinkClose

BinkClose 関数は、指定された Bink ハンドルを閉じます。

void BinkClose(
  HBINK bink
);

入力パラメータ

bink
閉じる Bink ハンドルを指定します。

説明

この関数は単に Bink ファイルを閉じて、関連するすべてのメモリを解放するだけです。開いたすべての Bink ファイルを閉じないと、システム リソースまたはメモリがリークすることがあります。


グループ: Bink API
関連項目: ニンテンドー ゲームキューブのリセット ボタンの取り扱い, ニンテンドー Wii のリセット ボタンの取り扱い, Open と Close
関連する関数: BinkGetSummary, BinkMacOpen, BinkNDSOpen, BinkOpen
関連する基本データ型: BINK_OPEN_FLAGS, HBINK, void

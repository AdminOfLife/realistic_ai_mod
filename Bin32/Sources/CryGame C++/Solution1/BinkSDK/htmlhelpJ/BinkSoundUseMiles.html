Bink SDK 1.9c

BinkSoundUseMiles

BinkSoundUseMiles 関数は、Miles Sound System をサウンド出力システムとして使用するように Bink に指示します。

S32 BinkSoundUseMiles(
  VOID_PTR hDigDriver
);

入力パラメータ

hDigDriver
Miles 関数 AIL_waveOutOpen が返す Miles Sound System の HDIGDRIVER ハンドルを指定します。

戻り値

Bink の出力レイヤが Miles に切り替わったかどうかを返します (正常に終了した場合は 0 以外、失敗した場合は 0)。

説明

この関数 (実際はマクロ) は、Miles Sound System をオーディオ出力として使用するように Bink に指示します。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前に呼び出さなければ、この関数は失敗します。

Miles を使用して、マルチチャンネル Bink ファイルを再生できるようになりました。詳細については、「Win32 でのサウンド」セクションまたは BinkSetMixBinVolumes 関数を参照してください。


グループ: Bink のオーディオ システムのインターフェイス関数
関連項目: Win32 でのサウンド, Win64 でのサウンド
関連する関数: BinkOpen, BinkSetMixBinVolumes, BinkSetSoundSystem, BinkSoundUseAX, BinkSoundUseDirectSound, BinkSoundUseLibAudio, BinkSoundUseMusyX, BinkSoundUseNDSSound, BinkSoundUsePSPSound, BinkSoundUseRAD_IOP, BinkSoundUseSDLMixer, BinkSoundUseSoundManager, BinkSoundUseWaveOut
関連する基本データ型: S32, VOID_PTR

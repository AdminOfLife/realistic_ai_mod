Bink SDK 1.9c

RAD_IOP_READCALLBACK

関数 RAD_IOP_READCALLBACK は、非同期の読み取りが完了したときに呼び出される関数をポイントします。

void RAD_IOP_READCALLBACK(
  RAD_IOP_ASYNC_READ_DATA const *read_data
);

入力パラメータ

read_data
RAD_IOPQueueAsyncReadAtTop または RAD_IOPQueueAsyncReadAtBottom を呼び出したときの構造体へのポインタを指定します。

説明

この関数は、読み取り完了時に (インタラプト レベルで) 1 回呼び出されます。この関数の中に作成するコードは、必ずインタラプト セーフにします。

このコールバックは成功と失敗のどちらの場合も実行されるため、ステータス フィールドの値を確認して、読み取りが実際に正常に行われたかどうかを判別してください。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPQueueAsyncReadAtBottom, RAD_IOPQueueAsyncReadAtTop
関連する基本データ型: void
関連する構造体: RAD_IOP_ASYNC_READ_DATA

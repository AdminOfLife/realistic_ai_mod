Bink SDK 1.9c

ニンテンドー Wii 用の Bink サンプル

説明

Bink SDK には、ニンテンドー Wii 用のプログラムのサンプルがいくつか用意されています。CodeWarrior を使用してこれらのサンプルを作成するには、新規プロジェクトを作成し、適切な C のソース ファイル (ExamWiiBB.C または ExamWiiTex.C) をインクルードし、次にニンテンドー Wii 版 Bink のスタティック ライブラリをプロジェクト (BinkWii.a) に追加するだけです。

練習用のサンプル ファイルは次のとおりです。

Bink を使用して、ビデオを直接 Wii の YUV バック バッファに表示するサンプルについては、ファイル ExamWiiBB.C を参照してください。このサンプルは、ビデオのレンダリングに GPU を使用しないため、きわめてシンプルです。バック バッファに直接ブラストするだけです。また、YUV フォーマットから変換する必要もないため、処理速度も格段に向上します。

ニンテンドー Wii の GPU ハードウェアを使用して、直接 Bink の内部 YUV バッファからレンダリングするサンプルについては、ExamWiiTex.C ファイルを参照してください。このサンプルでは 3D ラッパーを使用するので、必ずファイル WiiTextures.C もプロジェクトに追加してください。このサンプルは、通常の Bink ビデオまたはアルファ Bink ビデオをレンダリングします。

これらのサンプルでは、Wii を設定するために必ず 1 つの簡単なファイルを使用します。サンプルのプロジェクトには、WiiDemo.c も加えてください。

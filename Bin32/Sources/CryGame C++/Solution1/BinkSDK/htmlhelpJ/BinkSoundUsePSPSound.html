Bink SDK 1.9c

BinkSoundUsePSPSound

BinkSoundUsePSPSound 関数は、Sony PSP で libwave またはシンプル オーディオをサウンド出力システムとして使用するように Bink に指示します。

S32 BinkSoundUsePSPSound(
  U32 channel
);

入力パラメータ

channel
使用するサウンド チャンネルを指定します。

戻り値

Bink の出力レイヤが PSP サウンドに切り替わったかどうかを返します (正常に終了した場合は 0 以外、失敗した場合は 0)。

説明

この関数 (実際はマクロ) は、ソニーの livewave またはソニーのシンプル オーディオ システム API をオーディオ出力として使用するように Bink に指示します。

channel パラメータには、3 つの libwave チャンネルの 1 つを使用する場合は 0、1、または 2、シンプル オーディオを使用する場合は -1 を指定します。シンプル オーディオ API は、一度に 1 つのクライアントしか使用できません。したがって、シンプル オーディオを使用する場合は、最初にゲーム オーディオを停止する必要があります。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前に呼び出さなければ、この関数は失敗します。

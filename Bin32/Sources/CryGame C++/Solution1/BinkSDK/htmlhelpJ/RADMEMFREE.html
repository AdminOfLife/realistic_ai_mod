Bink SDK 1.9c

RADMEMFREE

RADMEMFREE 関数は、すでに RADMEMALLOC 関数を呼び出すことで割り当てられているメモリを解放する関数をポイントします。

void RADLINK RADMEMFREE(
  VOID_PTR ptr
);

入力パラメータ

ptr
RADMEMALLOC が直前に割り当てたポインタを指定します。

説明

この関数は、メモリを解放するために Bink が呼び出す関数を置き換えます。RADMEMALLOC が返すすべてのポインタを解放できる必要があります。


グループ: Bink API
関連する関数: BinkSetMemory, RADMEMALLOC
関連する基本データ型: VOID_PTR, void

Xbox 360 のサウンド

説明

Xbox 360 でサウンドをサポートするために、Bink は直接 XAudio に接続します。ユーザーのアクションは不要です。すべて自動的に処理されます。

5.1 チャンネル以外のサウンド トラックを再生する場合、Bink はデフォルトのステレオまたはモノラルのサウンド トラックを 5.1 チャンネルのすべてのスピーカーに接続します。

ただし、Bink を使用して 5.1 チャンネルのサウンド トラックを再生する場合は、ここでの手順に従って、Bink ファイルおよび再生コードを設定します。

5.1 チャンネルのサウンド トラックを再生するには、まず複数のオーディオ トラックを含む Bink ファイルを作成します (オーディオ トラックの詳細については、「Audio-tracks-in-Bink」を参照)。次に、Bink Sound Mixer を使用して各オーディオ トラックをミックスします。次のように、複数のサウンド トラックを作成します。

Track ID 0 - 左前方と右前方のチャンネルを含むサウンド トラック

Track ID 1 - 正面チャンネルを含むモノラル サウンド トラック

Track ID 2 - サブ ウーファーチャンネルを含むモノラル サウンド トラック

Track ID 3 - 左後方と右後方のチャンネルを含むサウンド トラック

そして、実行時にこれらすべてのサウンド トラックを一度に開くように Bink を設定します。それには、次のコードを使用します。

U32 TrackIDsToPlay[ 4 ] = { 0, 1, 2, 3 };
BinkSetSoundTrack( 4, TrackIDsToPlay );

bink = BinkOpen( "file.bik", BINKSNDTRACK );

次に、これらのサウンド トラックの出力先として DirectSound を指定します。それには、次のコードを使用します。

U32 bins[ 2 ];

bins[ 0 ] = XAUDIOSPEAKER_FRONTLEFT;
bins[ 1 ] = XAUDIOSPEAKER_FRONTRIGHT;
BinkSetMixBins( bink, 0, bins, 2 );

bins[ 0 ] = XAUDIOSPEAKER_FRONTCENTER;
BinkSetMixBins( bink, 1, bins, 1 );

bins[ 0 ] = XAUDIOSPEAKER_LOWFREQUENCY;
BinkSetMixBins( bink, 2, bins, 1  );

bins[ 0 ] = XAUDIOSPEAKER_BACKLEFT;
bins[ 1 ] = XAUDIOSPEAKER_BACKRIGHT;
BinkSetMixBins( bink, 3, bins, 2 );

ExamXenon.CPP には、これらを実行するサンプル コードが含まれています。詳細については、BINK51 コードの章を参照してください。

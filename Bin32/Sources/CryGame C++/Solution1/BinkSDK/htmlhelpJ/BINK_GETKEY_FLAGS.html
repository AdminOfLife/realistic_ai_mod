Bink SDK 1.9c

BINK_GETKEY_FLAGS

基本データ型 BINK_GETKEY_FLAGS で、どのようにキー フレームの検索を実行するかを指定します。

基本的な値

 値  説明
 BINKGETKEYPREVIOUS 指定されたフレームまたはそれよりも前にある最初のキー フレームを検索します。
 BINKGETKEYNEXT 指定されたフレームまたはそれよりも後にある最初のキー フレームを検索します。
 BINKGETKEYCLOSEST 指定されたフレームまたはそれに最も近いキー フレームを検索します (前と後ろの両方を検索します)。
 BINKGETKEYNOTEQUAL このフラグを他のフラグの 1 つと OR 演算で組み合わせることにより、指定したフレームと同じフレームを Bink がキー フレームとして返さないように指示します。

説明

これらのフラグを BinkGetKeyFrame 関数で使用して、キー フレームの検索を実行する方法を指定します。


グループ: Bink API
関連する関数: BinkGetKeyFrame

Bink SDK 1.9c

Win64 用の Bink サンプル

説明

Bink SDK には、Win64 用のサンプル プログラムがいくつか用意されています。IDE ベースの環境でこれらのサンプルを作成するには、新規プロジェクトを作成して適切な C のソース ファイル (ExamWin.C、ExamMSS.C など) をインクルードし、次に Windows 版 Bink のインポート ライブラリをプロジェクト (BinkW64.LIB) に追加するだけです。

練習用のサンプル ファイルは次のとおりです。

BinkBuffer API の簡単なサンプルについては、ExamWin.C ファイルを参照してください。

Miles Sound System と BinkBuffer API を使用するサンプルについては、ExamMSS.C ファイルを参照してください。

DirectDraw で Bink を使用するサンプルについては、ExamDDP.C ファイルを参照してください。

YUV 色変換を実行するために Bink でピクセル シェーダを使用するサンプルについては、Exam3Dps.CPP ファイルを参照してください。このサンプルは、元の Xbox、ソニー PS3、および Xbox 360 と互換性がある高レベルの D3D ラッパーを使用します。詳細については、BinkTextures.h ファイルと BinkTextures.cpp ファイルを参照してください。

Bink でピクセル シェーダを使用して、アルファ チャンネルを含むビデオを再生するサンプルについては、Exam3Dpsa.CPP ファイルを参照してください。このサンプルは、Exam3Dps.CPP と同じサポート ファイルを使用します。

Bink で固定機能 3D パイプラインを使用するサンプルについては、Exam3D.C ファイルを参照してください。このサンプルは、高レベルの固定機能パイプライン ラッパーを使用します。このサンプルをコンパイルして、DirectX 9、DirectX 8、DirectX 7、または OpenGL を使用できます。詳細については、RAD3D.h、DX9RAD3D.CPP、DX8RAD3D.CPP、DX7RAD3D.CPP、または GLRAD3D.C ファイルを参照してください。

Bink で固定機能 3D パイプラインを使用して、アルファ チャンネルを含むビデオを再生するサンプルについては、Exam3Da.CPP ファイルを参照してください。このサンプルは、Exam3D.C と同じサポート ファイルを使用します。

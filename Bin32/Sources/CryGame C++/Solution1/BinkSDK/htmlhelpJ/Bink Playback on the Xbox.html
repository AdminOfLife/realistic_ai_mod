Xbox での Bink 再生

説明

Xbox のスレッド サポートはたいへん優れており、一般的な Xbox の Bink 再生ループはとても単純です。Bink は BinkWait 関数でタイミングを処理するので、必要な処理は、ビデオの再生中にこの関数を確認することだけです。

Xbox での再生の簡単なサンプルは次のようになります。


while ( 1 ) 
{
  // フレームを解凍 
  BinkDoFrame( bink );


  // 必要に応じてフレームをスキップ...
  while ( BinkShouldSkip( bink )
  {
    BinkNextFrame( bink );
    BinkDoFrame( Bink );
  }


  // ここでビデオ テクスチャを描画するが、フレームはまだ表示しない
  Draw_Bink_textures( ... );


  // 入力の確認、他のタスクの処理などを実行
  while ( BinkWait( bink ) )
    do_idle();


  // バッファをフリップし、ビデオ フレームを表示
  d3d_device->Present();


  // ビデオが終了したか
  if ( bink->FrameNum == bink->Frames )
    break;

               
  // 次のフレームに移動
  BinkNextFrame( bink );  
}

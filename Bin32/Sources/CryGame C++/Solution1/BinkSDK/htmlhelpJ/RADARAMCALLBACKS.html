Bink SDK 1.9c

RADARAMCALLBACKS

RADARAMCALLBACKS 構造体は、ニンテンドー ゲームキューブまたは Wii で AX を使用するために代用するオーディオ RAM メモリ管理用の関数を指定します。

struct RADARAMCALLBACKS
{
  RADARAMALLOC mem_alloc;
  RADARAMFREE mem_free;
};

入力メンバ

mem_alloc
オーディオ メモリを割り当てるために呼び出す関数を指定します。
mem_free
mem_alloc が割り当てたオーディオ メモリを解放するために呼び出す関数を指定します。

説明

この構造体には、ニンテンドー ゲームキューブまたは Wii で AX を使用している場合に、Bink がオーディオ RAM を管理するために使用する割り当てと解放の関数に対するコールバックが含まれます。

実際には Wii に別のオーディオ RAM はありませんが、MEM2 からオーディオ メモリを割り当てる場合のために、別のメモリ割り当て関数を残してあります。


グループ: Bink のオーディオ システムのインターフェイス関数
関連する関数: BinkSoundUseAX, RADARAMALLOC, RADARAMFREE

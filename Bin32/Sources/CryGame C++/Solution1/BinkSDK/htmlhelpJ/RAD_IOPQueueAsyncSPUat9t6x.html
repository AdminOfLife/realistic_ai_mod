RAD_IOPQueueAsyncSPUGlobalState

関数 RAD_IOPQueueAsyncSPUGlobalState は、SPU コアの状態を非同期にクエリします。

S32 RAD_IOPQueueAsyncSPUGlobalState(
  RAD_IOP_SPU_GLOBAL_STATE *state_data
);

出力パラメータ

state_data
グローバル SPU 状態の入力パラメータと出力パラメータを指定します。

戻り値

クエリが正常にキューに置かれると 0 以外の値、失敗すると 0 を返します。

説明

この関数は、グローバル SPU クエリをキューに置き、バックグラウンドで実行されるようにします。

オプションで (またはゼロに) callback フィールドを設定し、この関数を呼び出します。

完了したことを確認するには、callback を使用するか、status フィールドを調べます。ステータスが RAD_IOP_STATUS_DONE になると、現在の状態フィールドは有効です。

このフィールドを使用すると、現在の再生位置、音声が再生中かどうか、および現在のエンベロープ レベルを確認できます。


グループ: ソニー PS2 用 RAD_IOP SPU ハードウェア音声 API

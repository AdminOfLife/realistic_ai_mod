Bink SDK 1.9c

BINKPLANE

BINKPLANE 構造体は、1 つの 2D 平面を記述する構造体です。

struct BINKPLANE
{
  S32 Allocate;
  VOID_PTR Buffer;
  U32 BufferPitch;
};

出力メンバ

Allocate
この平面を割り当てる必要があるかどうかを指定します。
Buffer
解凍先のアドレスを指定します (16 バイト境界で揃える必要があります)。
BufferPitch
1 つの走査線から次の走査線までのバッファのピッチをバイト単位で指定します。

説明

この構造体は、BINKFRAMEBUFFERS 構造体の最下位レベルの詳細情報です。これらの構造体の 1 つが各 Bink ビデオ平面を記述します (Y、cR、cB、およびオプションでアルファ平面ごとに 1 つ)。

Allocate フィールドは、この平面を割り当てる必要があるかどうかを指示します。たとえば、アルファ平面のないビデオの場合、BINKFRAMEPLANESET 構造体内の APlane 構造体で Allocate フィールドはゼロになります。

Buffer メモリ ポインタは、16 バイト境界で揃える必要があることに注意してください。

この構造体の詳しい使用法については、BinkGetFrameBuffersInfo 関数と BinkRegisterFrameBuffers 関数を参照してください。


グループ: Bink API
関連する関数: BinkGetFrameBuffersInfo, BinkRegisterFrameBuffers
関連する基本データ型: S32, U32, VOID_PTR
関連する構造体: BINKFRAMEBUFFERS, BINKFRAMEPLANESET

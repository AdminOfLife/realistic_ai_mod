Bink SDK 1.9c

GNU/Linux 用の Bink サンプル

説明

Bink SDK には、SDL ライブラリを使用した GNU/Linux 用のサンプルが用意されています。

このサンプルをビルドするには、次の手順に従います。
1) examsdl.c を GCC でコンパイルします。ビルドには SDL インクルード ディレクトリを使用する必要があります (sdl-config で自動的に実行できます)。
2) サンプルを examsdl オブジェクト ファイル、Bink ライブラリ (binklnx.a)、SDL と pthreads の共有ライブラリ、および当然ながら GCC glibc ライブラリにリンクします。

これで終了です。コマンド ラインでファイル名を入力すると、Bink サンプルを実行できます。

Bink SDK 1.9c

MacOS 用の Bink サンプル

説明

Bink SDK では、当社の Web サイトで提供している Bink プレイヤーのすべてのソース コードとリソースが提供されています。

これらのサンプルをビルドするには、次の手順に従います。
1) CodeWarrior で、Classic、Carbon、または Mach0 用 C プロジェクトの MacOS Toolbox を新しく作成します。
2) プロジェクト内に、player.c、player.r、playerbinary.rsrc の各ファイルを追加します。Carbon のプロジェクトの場合は、carbon.h ファイルもプロジェクトに追加します。
3) 適切な Bink ライブラリ (BinkClassicLibrary、BinkCarbonLibrary、libBinkMachO.dylib、libBinkMachOx86.dylib のいずれか) を実行可能ファイルのディレクトリにコピーします。MachO アプリケーションでは、アプリケーションの実行可能ファイルが Contents/MacOS フォルダに置かれていることに注意してください。MachO アプリケーション パッケージは、単なる定義済みフォルダ セットです。したがって、Finder で Contents サブフォルダを開くには、Control キーを押しながらアプリケーションのアイコンをクリックし、[Show Package Contents] を選択します。次に、libBinkMachO.dylib または libBinkMach0x86.dylib ファイルを Contents/MacOS フォルダに入れます。
4) コピーした Bink ライブラリをプロジェクトに追加します。
5) CodeWarrior がデフォルトでプロジェクト内に置いたサンプルのソース ファイルを削除します。

これで終了です。当社が再配布したものと同じ Bink サンプルをビルドして実行できるようになります。

Bink SDK 1.9c

アルファ平面

説明

アルファ平面は、ビデオ フレーム上の要素が透過する、つまり透明であるかどうかを記述するビット平面です。この平面は透過性をピクセル単位で制御できるため、単なるカラーキーよりもはるかに便利で、カラーキーを非透過ピクセルのエッジにブレンドしたときの面倒な問題が発生しません。

アルファ平面を作成するには、Adobe Premiere などのビデオ編集パッケージを使用するか、ビデオ フレームのレンダリング時に 3D パッケージにアルファ平面の生成を指定します。

ピクセルのアルファ データを保存する方法には、プリマルチプライド アルファと通常のアルファの 2 つの方法があります。必ずプリマルチプライド アルファを使用してください。これは、通常のアルファより高速で圧縮率が高く、3D ハードウェアで正しくフィルタ処理されます。

アルファ透過が適用されると CPU をかなり使用するため、アルファ平面の再生には多少の工夫が必要です。高いパフォーマンスを維持するには、3D ビデオ カードを使用して、アルファ平面を使用したビットマップをマージすることをお勧めします。

3D ビデオ カードを使用すると、OpenGL、Direct3D、またはコンソールの 3D ライブラリを使用してビデオ カードを操作することが必要になります。

Win32 および Win64 の Bink には、ピクセル シェーダ ("Exam3DPSa.CPP") を使用して Direct3D のアルファ透過ビデオを再生できるサンプルが含まれます。このサンプルは、アプリケーションにドロップするだけのサンプル ソース コード (BinkTextures.h および BinkTextures.CPP) を使用しています。

Win32 および Win64 の Bink には、OpenGL と Direct3D の両方に対応する 3D 固定機能パイプラインを使用してアルファ透過ビデオを再生できるサンプルも含まれます (Exam3Da.C)。OpenGL または Direct3D を使用するには、該当する 3D インターフェイス ファイル (GLRAD3D.C、DX9RAD3D.CPP、DX8RAD3D.CPP、または DX7RAD3D.CPP) にリンクします。

Xbox の Bink には、Direct3D を使用してアルファ透過ビデオを再生するサンプルが含まれます (ExamXAlf.CPP)。Xbox 3D インターフェイス ファイル BinkTextures.CPP も必要です。

Xbox 360 の Bink には、Direct3D を使用してアルファ透過ビデオを再生するサンプルが含まれます (ExamXenA.CPP)。Xbox 360 3D インターフェイス ファイル BinkTextures.CPP も必要です。

ソニー PS3 の Bink には、PSGL を使用してアルファ透過ビデオを再生するサンプルが含まれます (ExamPS3A.CPP)。GL 3D インターフェイス ファイル PSGLTextures.CPP も必要です。

ソニー PSP の Bink には、libGu を使用してアルファ透過ビデオを再生するサンプルが含まれます (ExamPSP.CPP)。Gu 3D インターフェイス ファイル BinkGuTextures.CPP も必要です。

ニンテンドー Wii の Bink には、GPU を使用してアルファ透過ビデオを再生するサンプルが含まれます (ExamWiiTex.C)。Wii 3D インターフェイス ファイル WiiTextures.C も必要です。

ニンテンドー ゲームキューブの Bink には、GPU を使用してアルファ透過ビデオを再生するサンプルが含まれます (ExamNGCtex.C)。ゲームキューブ 3D インターフェイス ファイル "NGCTextures.C" も必要です。

Windows、ソニー PS3、Wii、ゲームキューブ、Xbox、および Xbox 360 でピクセル シェーダ/レジスタ コンバイナ アルファ ブリッティングを実行するためのクロス プラットフォーム API が用意されています。詳細については、BinkTextures.h ファイルを参照してください。


重要事項 3D システムで非アルファ平面ビデオを再生する場合は、アルファ ブレンディングのレンダリング状態をオフにする必要があります。Bink は、非アルファ ピクセル フォーマット (BINKSURFACE32 など) でアルファ値を設定しません。設定すると、ブリッティング速度が遅くなるためです。非アルファ ビデオを再生しようとして、画面が黒くなる場合は、アルファのレンダリング状態がオンになっている可能性があります。

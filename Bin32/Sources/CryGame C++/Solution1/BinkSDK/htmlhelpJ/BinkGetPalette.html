Bink SDK 1.9c

BinkGetPalette

BinkGetPalette 関数は、BINKSURFACEP8 サーフェス タイプのグローバル Bink パレットを返します。

void BinkGetPalette(
  VOID_PTR dest_addr
);

入力パラメータ

dest_addr
グローバル Bink パレットを取得するための 1024 バイトのアドレスです。

説明

この関数は、BINKSURFACEP8 サーフェス タイプをディザリングするために Bink が使用するグローバル パレットを返します。このパレットは、常に単純なフォーマット (B バイト、G バイト、R バイト、パッド バイトなど) で返されます。


グループ: Bink API
関連する基本データ型: BINK_COPY_FLAGS, VOID_PTR, void

Bink SDK 1.9c

BinkSetSoundSystem

BinkSetSoundSystem 関数は、カスタムのサウンド プロバイダをサウンド出力システムとして使用するように Bink に指示します。

S32 BinkSetSoundSystem(
  VOID_PTR BinkOpenSoundSystemFunction,
  U32 param_value
);

入力パラメータ

BinkOpenSoundSystemFunction
カスタムのサウンド システムのオブジェクトを開くために呼び出す関数を指定します。
param_value
OpenSoundSystemFunction 関数に渡す 32 ビットのサウンド システムのパラメータを指定します。

戻り値

Bink から新しいサウンド システムに正常に切り替えられたかどうかを返します。

説明

この関数は、Bink で新しいサウンド システムに切り替えるための低レベルの方法です。BinkSoundUseMiles と BinkSoundUseDirectSound は、両方ともこの関数を呼ぶ出すだけの単純なマクロです。

この関数を使用して独自のサウンド システム プロバイダを開発できますが、サウンド プロバイダを開発することは大がかりな作業であり、必要になることはほとんどありません。それでも独自のプロバイダの開発に関心がある場合は、詳細について RAD にお問い合わせください。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前に呼び出さなければ、この関数は失敗します。


グループ: Bink のオーディオ システムのインターフェイス関数
関連する関数: BinkOpen, BinkSoundUseAX, BinkSoundUseDirectSound, BinkSoundUseLibAudio, BinkSoundUseMiles, BinkSoundUseMusyX, BinkSoundUseNDSSound, BinkSoundUsePSPSound, BinkSoundUseRAD_IOP, BinkSoundUseSDLMixer, BinkSoundUseSoundManager, BinkSoundUseWaveOut

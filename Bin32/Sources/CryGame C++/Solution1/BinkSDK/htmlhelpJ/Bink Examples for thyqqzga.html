Bink SDK 1.9c

ソニー PS2 用の Bink サンプル

説明

Bink SDK には、ソニー PS2 用のプログラムのサンプルがいくつか用意されています。SN システムや CodeWarrior でこれらのサンプルを作成するには、新規プロジェクトを作成して適切な C のソース ファイル (ExamPbb.C、または ExamPTex.C) をインクルードし、次にソニー PS2 版 Bink のスタティック ライブラリをプロジェクト (BinkPS2.a) に追加するだけです。

練習用のサンプル ファイルは次のとおりです。

Bink を使用して、ビデオを直接プレイステーションのバック バッファに表示するサンプルについては、ファイル ExamPbb.C を参照してください。このサンプルはきわめて単純です。EE を使用してフレームを解凍し、RGB32 メモリ バッファに変換しています。次に、DMA システムを使用して RGB バッファを PS2 バック バッファにコピーします。変換が完了したら、ページが次のバーティカル ブランクにフリップするように要求します。これらはすべて PS2 の標準に従っています。

Bink を使用して、テクスチャに解凍してからテクスチャを画面上にレンダリングするサンプルについては、ファイル ExamPTex.C を参照してください。このサンプルでは多少複雑なコードを使用して、各ビデオ フレームを RGB32 テクスチャにアップロードしてから画面にレンダリングしています。これは、実際の Bink コードよりも PS2 のインターフェイス コードを使用するための単純なサンプルでもあります。

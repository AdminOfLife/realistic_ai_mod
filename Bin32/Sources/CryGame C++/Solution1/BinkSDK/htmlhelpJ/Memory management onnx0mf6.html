ニンテンドー DS のメモリ管理

説明

デフォルトでは、Bink は OS のメモリ割り当て関数を直接呼び出して、メモリ管理を処理します。ただし、Bink の割り当て関数や解放関数を BinkSetMemory 関数に置き換えることもできます。

Bink ムービーにサウンド トラックがある場合、BinkDoFrame 関数は 32 KB のスタック領域を使用します。スタック フレームがこれを処理できるほど大きいことを確認してください。

BinkGetSummary 関数を使用すると、メモリの使用量や最大使用量を確認できます。

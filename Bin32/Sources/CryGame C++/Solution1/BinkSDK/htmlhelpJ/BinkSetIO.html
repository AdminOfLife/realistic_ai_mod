Bink SDK 1.9c

BinkSetIO

BinkSetIO 関数は、ファイル I/O の関数の代用を指定します。

void BinkSetIO(
  VOID_PTR BinkOpenFileFunction
);

入力パラメータ

BinkOpenFileFunction
Bink ファイルを開くために呼び出す関数を指定します。

説明

この関数を使用すると、Bink の I/O システム全体を置き換えることができます。これは明らかに大がかりな作業ですが、これによって Bink はすべての種類のデバイスから読み込むことができるようになります。

この関数が必要になることは、ほとんどありません。圧縮されたファイルを読み込む場合は、BINKFILEHANDLE フラグを使用してください。そのほうがはるかに簡単です。

それでもこの機能に関心がある場合は、RAD Game Tools にご連絡ください。標準ファイル システムの I/O プロバイダのソース コードを参考として差し上げます。

この関数を使用するには、最初にこの関数を呼び出して、次に BINKIOPROCESSOR フラグを BinkOpen の呼び出しに追加します。


グループ: Bink API
関連項目: ソニー PS2 でのファイル I/O
関連する関数: BinkOpen
関連する基本データ型: BINK_OPEN_FLAGS, VOID_PTR, void

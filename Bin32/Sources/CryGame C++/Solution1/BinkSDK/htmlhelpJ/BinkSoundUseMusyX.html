Bink SDK 1.9c

BinkSoundUseMusyX

BinkSoundUseMusyX 関数は、ニンテンドー ゲームキューブでのサウンド出力システムとして MusyX を使用するように Bink に指示します。

S32 BinkSoundUseMusyX();

戻り値

Bink の出力レイヤが MusyX に切り替わったかどうかを返します (正常に終了した場合は 0 以外、失敗した場合は 0)。

説明

この関数 (実際はマクロ) は、ニンテンドー ゲームキューブでのオーディオ出力に MusyX を使用するように Bink に指示します。AX を使用している場合は、代わりに BinkSoundUseAX を使用します。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前に呼び出さなければ、この関数は失敗します。


グループ: Bink のオーディオ システムのインターフェイス関数
関連項目: ニンテンドー ゲームキューブのサウンド
関連する関数: BinkOpen, BinkSetSoundSystem, BinkSoundUseAX, BinkSoundUseDirectSound, BinkSoundUseLibAudio, BinkSoundUseMiles, BinkSoundUseNDSSound, BinkSoundUsePSPSound, BinkSoundUseRAD_IOP, BinkSoundUseSDLMixer, BinkSoundUseSoundManager, BinkSoundUseWaveOut
関連する基本データ型: S32

RAD_IOP_STATUS_VALUES

基本データ型 RAD_IOP_STATUS_VALUES は、RAD_IOP 非同期 I/O 関数のステータス値を指定します。

基本的な値

 値  説明
 RAD_IOP_STATUS_NONE ステータスが設定されていません。
 RAD_IOP_STATUS_QUEUED 要求は、バックグラウンドでの処理のためにキューに置かれます。
 RAD_IOP_STATUS_RUNNING 要求は、現在バックグラウンドで処理されています。
 RAD_IOP_STATUS_COMPLETE ステータスが RAD_IOP_STATUS_COMPLETE 以上の場合、バックグラウンド演算は完了しています。
 RAD_IOP_STATUS_CANCELLED この要求に対して、非同期 I/O 取り消し関数の 1 つが呼び出されました。
 RAD_IOP_STATUS_ERROR バックグラウンド演算に失敗しました (オープン時にファイルが見つからないか読み取り時の I/O エラー)。
 RAD_IOP_STATUS_DONE バックグラウンド演算が正常に完了しました。

説明

これらのフラグは、RAD_IOP の非同期のオープン関数と読み取り関数で使用されます。バックグラウンド演算の現在のステータスを示します。


グループ: ソニー PS2 用 RAD_IOP API

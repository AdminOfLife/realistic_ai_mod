Bink SDK 1.9c

BinkSetSimulate

BinkSetSimulate 関数は、指定されたデバイスへの転送速度をシミュレートするように Bink に指示します。

void BinkSetSimulate(
  U32 sim_speed
);

入力パラメータ

sim_speed
シミュレートする速度を 1 秒あたりのバイト数で指定します。

説明

この関数を使用すると、再生速度をシミュレートできます。これは、Bink のムービーが特定の速度の CD-ROM でどのように再生されるかを確認する場合に便利です。この値は 1 秒あたりのバイト数で設定するので、等倍速 CD では 150,000、2 倍速 CD では 300,000、4 倍速 CD では 600,000 を使用します。

この関数を使用するには、最初にこの関数を呼び出して、次に BINKSIMULATE フラグを BinkOpen の呼び出しに追加します。


グループ: Bink API
関連する関数: BinkOpen
関連する基本データ型: BINK_OPEN_FLAGS, U32, void

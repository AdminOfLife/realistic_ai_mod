Bink SDK 1.9c

BinkSetPan

BinkSetPan 関数は、Bink のサウンド トラックのスピーカーのパン (右対左のバランス) を設定します。

void BinkSetPan(
  HBINK bink,
  U32 trackID,
  S32 pan
);

入力パラメータ

bink
Bink ハンドルを指定します。
trackID
パンを変更するトラック ID を指定します。
pan
パンの新しい設定を指定します。

説明

この関数で、現在再生している Bink のサウンド トラックのスピーカーのパン (左右のバランス) を設定します。

パンのパラメータには、左右のバランスの新しい設定を 0 (左だけ) から 32768 (左右が均等)、65536 (右だけ) の範囲で指定します。

Xbox または Xbox 360 ではパンを設定できません。これらのプラットフォームでパンを制御するには、BinkSetMixBins 関数と BinkSetMixBinVolumes 関数を使用します。

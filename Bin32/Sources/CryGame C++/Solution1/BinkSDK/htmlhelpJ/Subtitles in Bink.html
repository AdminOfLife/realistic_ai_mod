Subtitles in Bink

Discussion

Bink doesn't support subtitles directly, but it's playback architecture makes them super easy to add. Here's how to use them:

1) For each Bink file in your application, add text files that contain the subtitles. So, if you had a video called example.bik, you create text files such as example.bik.eng.txt, example.bik.ger.txt, etc.

2) In each text file, on each line, place a start and a stop frame number, and then the text to display.

3) Then, at runtime, whenever you successfully open a Bink file, attempt to load the text file for the locale you are currently running on. To load the file, just open it and read the text into a std::map (or whatever).

4) Inside the standard Bink playback loop, look at the bink->FrameNum field and if you find a matching range in the map, display the text onto the screen using your normal application font.

And that's it. Nice and simple, no special tools are necessary, the subtitle font will match the rest of your application, and you can farm out the subtitling work without complicated instructions.

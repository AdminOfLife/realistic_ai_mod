Bink SDK 1.9c

RAD_IOPPauseTimer

関数 RAD_IOPPauseTimer は、すでに登録されているタイマー コールバックを中断します。

S32 RAD_IOPPauseTimer(
  RAD_IOP_TIMER_DATA *timer_data
);

出力パラメータ

timer_data
制御するタイマーを指定します。

戻り値

正常ならば 0 以外の値、失敗の場合は 0 を返します。

説明

この関数は、RAD_IOP に、タイマー コールバックの呼び出しを停止するように指示します。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPStartTimer
関連する基本データ型: S32
関連する構造体: RAD_IOP_TIMER_DATA

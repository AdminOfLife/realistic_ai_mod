Bink SDK 1.9c

BinkSetMixBinVolumes

BinkSetMixBinVolumes 関数は、Xbox、Xbox 360、ソニー PS3、または Miles Sound System の個々のミックス ビンの音量を設定します。

void BinkSetMixBinVolumes(
  HBINK bink,
  U32 trackID,
  u32_ptr mix_bins,
  s32_ptr volumes,
  U32 total
);

入力パラメータ

bink
Bink ハンドルを指定します。
trackID
ミックス ビンの音量を変更するトラック ID を指定します。
mix_bins
各音量に対応するミックス ビン定数の配列をポイントします (Miles または PS3 の場合は 0 を渡します)。
volumes
新しい音量レベルの配列をポイントします。
total
mix_bins 配列と volumes 配列内のミックス ビンの数。

説明

この関数を使用すると、Xbox、Xbox 360、ソニー PS3、または Miles Sound System で、各ミックス ビンの出力先に対して音量を設定できます。

ミックス ビンの出力先の値については、Xbox または Xbox 360 のドキュメントを参照してください。ソニー PS3 および Miles 7 の場合、音量配列はスピーカーのリニア配列です。ミックス ビン定数を渡す必要はないため、mix_bins パラメータには 0 を渡します。

volume パラメータには、0 (無音) から 32768 (フル音量) までの新しい音量設定を指定します。

この関数を使用してマルチチャンネルのサウンド再生を有効にする最適な方法については、Xbox、PS3、または Miles サンプル プログラムを参照してください。

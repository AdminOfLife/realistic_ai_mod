Bink SDK 1.9c

RAD_IOPStartSound

関数 RAD_IOPStartSound は、すでに登録されているサウンド コールバックの呼び出しを開始します。

S32 RAD_IOPStartSound(
  RAD_IOP_SOUND_DATA *sound_data
);

出力パラメータ

sound_data
制御するサウンド コールバックを指定します。

戻り値

正常ならば 0 以外の値、失敗の場合は 0 を返します。

説明

この関数は、RAD_IOP に、サウンド コールバックの呼び出しを開始するように指示します。呼び出されるたびに、最短時間でアドレスをサウンド データでいっぱいにして戻るようにしてください。

コールバックはインタラプト時に実行されるため、コールバックでは必ずインタラプト セーフ コードだけを使用してください。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPPauseSound, RAD_IOPRegisterSoundCallback
関連する基本データ型: S32
関連する構造体: RAD_IOP_SOUND_DATA

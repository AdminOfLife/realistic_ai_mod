Bink SDK 1.9c

RADMEMALLOC

RADMEMALLOC 関数は、Bink にメモリを割り当てる関数をポイントします。

VOID_PTR RADLINK RADMEMALLOC(
  U32 num_bytes
);

入力パラメータ

num_bytes
割り当てるメモリをバイト数で指定します。

戻り値

新しいポインタ、または割り当てに失敗した場合は 0 か -1 を返します。

説明

この関数は、メモリを割り当てるために Bink が呼び出す関数を置き換えます。num_bytes バイト以上のメモリへのポインタを返す必要があります。

割り当てが失敗した場合、0 を返すと Bink はオペレーティング システムからメモリの割り当てを試みますが、-1 を返すと Bink はそのまま割り当てに失敗します。

この関数からは、32 バイト境界で揃えられたメモリ ポインタを返す必要があります。

関連する関数: BinkSetMemory, RADMEMFREE

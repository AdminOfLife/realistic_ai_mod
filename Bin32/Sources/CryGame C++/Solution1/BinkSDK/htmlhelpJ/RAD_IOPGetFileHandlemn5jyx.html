Bink SDK 1.9c

RAD_IOPGetFileHandleToIOPMemory

関数 RAD_IOPGetFileHandleToIOPMemory は、IOP メモリからの読み取りに使用できるファイル ハンドルを作成します。

U32 RAD_IOPGetFileHandleToIOPMemory(
  VOID_PTR IOP_address
);

入力パラメータ

IOP_address
読み取りを開始する IOP メモリのアドレスを指定します。

戻り値

IOP メモリへのファイル ハンドル。

説明

この関数は、IOP メモリからの非同期の読み取りに使用できるファイル ハンドルを作成します。

データを IOP から SPU にコピーするには、このようなファイル ハンドルを使用し、適切なコピー先ポインタ型を指定して、非同期読み取り関数を使用します。

この種類のファイル ハンドルを閉じる必要はありません。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPQueueAsyncOpenFile, RAD_IOPQueueAsyncReadAtBottom, RAD_IOPQueueAsyncReadAtTop
関連する基本データ型: U32, VOID_PTR

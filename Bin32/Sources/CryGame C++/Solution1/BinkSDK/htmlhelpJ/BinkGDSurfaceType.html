Bink SDK 1.9c

BinkGDSurfaceType

関数 BinkGDSurfaceType は、指定された MacOS のグラフィック デバイスのフォーマットを表す値を返します。

BINK_COPY_FLAGS BinkGDSurfaceType(
  GDHandle gdevice
);

入力パラメータ

gdevice
クエリするグラフィック デバイスを指定します。

戻り値

指定されたサーフェスのフォーマットを表す値を返します。

説明

この関数は、提供されたグラフィック デバイスを表す値を返します。この関数を使用してサーフェス タイプを取得し、BinkCopyToBuffer 関数に渡すことができます。

サーフェスのタイプが未定義またはサポートされていない場合、この関数は -1 を返します。

Win32 では、BinkDDSurfaceType、BinkDX8SurfaceType、または BinkDX9SurfaceType 関数を使用します。


グループ: BinkBuffer API
関連する関数: BinkCopyToBuffer, BinkCopyToBufferRect, BinkDDSurfaceType, BinkDX8SurfaceType, BinkDX9SurfaceType
関連する基本データ型: BINK_COPY_FLAGS, GDHandle

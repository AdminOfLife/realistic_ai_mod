RAD_IOPShutDown

関数 RAD_IOPShutDown は、RAD_IOP ライブラリを閉じます。

S32 RAD_IOPShutDown();

戻り値

ライブラリが閉じたら 0 以外の値、RAD_IOPShutDown が RAD_IOPStartUp の呼び出し回数と同じ回数だけ呼び出されていないために開いたままならば 0 を返します。

説明

この関数は RAD_IOP ライブラリを閉じます。進行中のすべての I/O を取り消してから、開始されていた IOP IRX を終了します。その後、ユーザーは、すでに RAD_IOPStartUp に渡してある IOP メモリを解放できます。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPMemoryAmount, RAD_IOPShutDown, RAD_IOPStartUp
関連する基本データ型: S32

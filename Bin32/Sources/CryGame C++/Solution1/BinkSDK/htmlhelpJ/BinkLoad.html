Bink SDK 1.9c

BinkLoad

BinkLoad 関数は、Xbox 上にある Bink のセクションを (BinkUnload でアンロードされていた場合) リロードします。

void BinkLoad();

説明

BinkUnload 関数で Bink をアンロードした場合は、この関数でメモリにリロードします。デフォルトでは、アプリケーションの起動時に Bink とそのすべての YUV 変換がロードされます。この関数を呼び出す必要がある場合は、その前にアンロードする必要があります。


グループ: Bink API
関連項目: Xbox のメモリ管理, Xbox のセクション
関連する関数: BinkLoadConverter, BinkUnload
関連する基本データ型: void

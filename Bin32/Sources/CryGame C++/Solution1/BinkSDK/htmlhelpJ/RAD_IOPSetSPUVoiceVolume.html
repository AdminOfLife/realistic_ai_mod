RAD_IOPSetSPUVoiceVolume

関数 RAD_IOPSetSPUVoiceVolume は、1 つ以上の SPU 音声の音量を設定します。

S32 RAD_IOPSetSPUVoiceVolume(
  U32 core0_voice_mask,
  U32 core1_voice_mask,
  S32 left,
  S32 right
);

入力パラメータ

core0_voice_mask
設定するコア 0 の音声のビット マスクを指定します。
core1_voice_mask
設定するコア 1 の音声のビット マスクを指定します。
left
左スピーカーに 0〜0x3fff の音量を指定します。変更しない場合は -1 を指定します。
right
右スピーカーに 0〜0x3fff の音量を指定します。変更しない場合は -1 を指定します。

戻り値

正常ならば 0 以外の値、失敗の場合は 0 を返します。

説明

この関数は、1 つ以上の SPU 音声の基準音量を設定するように SPU に指示します。

最終的な音量レベルは、エンベロープの設定とマスタ音量レベルによって大きく左右されます。


グループ: ソニー PS2 用 RAD_IOP SPU ハードウェア音声 API
関連する関数: RAD_IOPSetSPUVoiceADSR
関連する基本データ型: S32, U32

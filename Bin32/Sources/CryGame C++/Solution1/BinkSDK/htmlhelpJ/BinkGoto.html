Bink SDK 1.9c

BinkGoto

BinkGoto 関数は、Bink ファイル内の新しいフレームに移動します。

void BinkGoto(
  HBINK bink,
  U32 frame_num,
  BINK_GOTO_FLAGS goto_flags
);

入力パラメータ

bink
Bink ハンドルを指定します。
frame_num
ジャンプ先のフレーム番号を指定します。
goto_flags
実行する移動の種類を指定します。

説明

この関数を使用すると、Bink ファイル内の他のフレームに直接スキップできます。パラメータ frame_num は、スキップ先のフレームを指定します (先頭のフレームがフレーム 1)。新しいフレームのデータが Bink の先読みメモリ内にある場合、この関数はただちに実行されます (いくつかのポインタが更新されるだけです)。

しかし、新しいフレームのデータが Bink の読み込みバッファ上にない場合は、Bink は検索を実行し (CD-ROM の場合は低速)、フレームのデータを読み込みます。ただし、BinkGoto は検索が必要な場合もきわめて高速です。検索のために一時停止したことに気付かないことも珍しくありません。

BinkGetKeyFrame 関数を使用して、最も近いキー フレームを手動で検索することもできます。

関連する関数: BinkGetKeyFrame
関連する基本データ型: BINK_GOTO_FLAGS, HBINK, U32, void

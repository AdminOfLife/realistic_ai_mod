Bink SDK 1.9c

BinkSetSoundOnOff

BinkSetSoundOnOff 関数は、オーディオの解凍を禁止し、オーディオのリソースを解放するように Bink に指示します。

S32 BinkSetSoundOnOff(
  HBINK bink,
  S32 on_off
);

入力パラメータ

bink
Bink ハンドルを指定します。
on_off
サウンド トラックを処理するか (0 以外)、または Bink がオーディオの解凍をスキップするかを指定します。

戻り値

サウンドが現在オンか (0 以外) オフか (0) を返します。

説明

この関数を使用すると、Bink ファイルのオーディオの再生を中止できます。すべてのサウンドを完全に停止したり、wave ファイルや AVI ファイルを再生するためにデジタル サウンドのリソース ハンドルを解放する必要がある場合に便利です。

この関数の機能はミュートではないことに注意してください。オーディオをミュートする場合は、音量レベルを 0 に設定して BinkSetVolume 関数を呼び出します。

オーディオを再生することが絶対にないとわかっている場合は、この関数を使用しません。代わりに、BinkSetSoundTrack 関数の両方のパラメータを 0 にして呼び出してから、BINKSNDTRACK フラグを指定して Bink ファイルを開きます。

この関数はトグルなので、一度 0 で呼び出してサウンドをオフにし、その後 0 以外の値で呼び出してオーディオの再生を再開します。

この関数は、オーディオがオンの場合は 0 以外、オーディオがオフの場合は 0 を返します。

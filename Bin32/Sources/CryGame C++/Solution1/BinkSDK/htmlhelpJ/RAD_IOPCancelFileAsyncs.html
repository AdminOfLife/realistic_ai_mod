RAD_IOPCancelFileAsyncs

関数 RAD_IOPCancelFileAsyncs は、指定されたファイル ハンドルによる今までのすべての非同期ファイル読み取りを取り消します。

S32 RAD_IOPCancelFileAsyncs(
  S32 file_handle
);

入力パラメータ

file_handle
読み取り要求を取り消すファイルのハンドルを指定します。

戻り値

取り消された非同期処理の数を返します (0 の場合、通常はすでに取り消しが完了していることを示します)。

説明

この関数を使用すると、特定のファイル ハンドルに基づいて、すでにキューに置かれているすべての読み取り要求を取り消すことができます。ただし、コールバックはオープン要求が取り消されたときも実行されるため、必ずステータス フィールドを確認してください。


グループ: ソニー PS2 用 RAD_IOP API
関連する関数: RAD_IOPCancelAllAsyncs, RAD_IOPCancelAsyncRead, RAD_IOPCancelGroupOfAsyncs
関連する基本データ型: S32

Bink SDK 1.9c

BINKFRAMEPLANESET

BINKFRAMEPLANESET 構造体は、平面のセット (Y、cR、cB、およびアルファに対して 1 つ) を記述する構造体です。

struct BINKFRAMEPLANESET
{
  BINKPLANE YPlane;
  BINKPLANE cRPlane;
  BINKPLANE cBPlane;
  BINKPLANE APlane;
};

出力メンバ

YPlane
Y 平面の詳細を指定します。
cRPlane
cR 平面の詳細を指定します。
cBPlane
cB 平面の詳細を指定します。
APlane
アルファ平面の詳細を指定します。

説明

この構造体は、BINKFRAMEBUFFERS 構造体内で使用されます。Bink の内部バッファ上の 1 つのビデオ フレームを表します。Y、cR、cB に 1 つずつとオプションでアルファ平面に 1 つの合計 4 つの詳細セットがあります。

この構造体の詳しい使用法については、BinkGetFrameBuffersInfo 関数と BinkRegisterFrameBuffers 関数を参照してください。


グループ: Bink API
関連する関数: BinkGetFrameBuffersInfo, BinkRegisterFrameBuffers
関連する構造体: BINKFRAMEBUFFERS, BINKPLANE

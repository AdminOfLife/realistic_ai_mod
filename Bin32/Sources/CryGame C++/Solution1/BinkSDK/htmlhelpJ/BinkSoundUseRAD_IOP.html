Bink SDK 1.9c

BinkSoundUseRAD_IOP

BinkSoundUseRAD_IOP 関数は、RAD_IOP API をサウンド出力システムとして使用するように Bink に指示します (PS2 の IOP チップを除く)。

S32 BinkSoundUseRAD_IOP(
  U32 which_sound_core
);

入力パラメータ

which_sound_core
0 または 1 を渡して、サウンド コア 0 またはサウンド コア 1 を使用するように Bink に指示します。

戻り値

Bink の出力レイヤが RAD_IOP に切り替わったかどうかを返します (正常に終了した場合は 0 以外、失敗した場合は 0)。

説明

この関数 (実際はマクロ) は、PS2 固有の RAD_IOP API をオーディオ出力として使用するように Bink に指示します。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前で RAD_IOPStartUp を呼び出した後に呼び出さなければ、この関数は失敗します。

Bink が使用するサウンド コアでデフォルトのマスタ音量レベルをあらかじめ確立していない場合、RAD_IOPHardwareVolumes を呼び出さないとオーディオは聞こえません。


グループ: Bink のオーディオ システムのインターフェイス関数
関連項目: ソニー PS2 のサウンド
関連する関数: BinkOpen, BinkSetSoundSystem, BinkSoundUseAX, BinkSoundUseDirectSound, BinkSoundUseLibAudio, BinkSoundUseMiles, BinkSoundUseMusyX, BinkSoundUseNDSSound, BinkSoundUsePSPSound, BinkSoundUseSDLMixer, BinkSoundUseSoundManager, BinkSoundUseWaveOut, RAD_IOPHardwareVolumes, RAD_IOPStartUp
